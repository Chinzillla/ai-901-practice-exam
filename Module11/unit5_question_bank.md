### 1. A response has the right facts but the wrong format. Which strategy should usually be tried first?

A. RAG  
B. fine-tuning  
C. combined strategy  
D. Prompt engineering  

**Answer:** D  
**Rationale:** The requirement matches the unit capability named in the correct option.

### 2. What is the main purpose of "prompt engineering"?

A. adapting model behavior with additional training examples  
B. using multiple optimization approaches together when one is not enough  
C. improving responses by changing instructions and examples  
D. improving factual grounding by retrieving relevant source content  

**Answer:** C  
**Rationale:** "prompt engineering" is a core idea in this unit, and the correct option paraphrases its role.

### 3. A model needs answers from frequently updated support documents. Which strategy fits best?

A. Information extraction  
B. Retrieval augmented generation  
C. Object detection  
D. Speech synthesis  

**Answer:** B  
**Rationale:** The correct option is the unit concept that best fits the stated requirement.

### 4. How should teams choose among prompting, RAG, and fine-tuning?

A. Match the strategy to the problem: instructions, missing knowledge, or consistent learned behavior.  
B. Always fine-tune before trying prompts.  
C. Use RAG only to change response tone.  
D. Use prompting to update private documents automatically.  

**Answer:** A  
**Rationale:** The unit teaches comparing strategies by problem type.

### 5. What should a learner remember about "RAG"?

A. a configuration task for the interface rather than the AI behavior  
B. a rule-based process that does not use model reasoning  
C. an access-control setting rather than the concept being defined  
D. making factual grounding by retrieving relevant source content better  

**Answer:** D  
**Rationale:** "RAG" is tested by recognizing its purpose, not by memorizing wording from the page.

### 6. Why might a solution combine RAG with prompt engineering?

A. Prompting prevents retrieved content from being used.  
B. RAG changes model weights automatically.  
C. RAG supplies trusted context, while prompting controls how the answer should be produced.  
D. The two strategies cancel each other out.  

**Answer:** C  
**Rationale:** Optimization strategies can complement each other.

### 7. Which two problems are good fits for prompt engineering? Choose two.

A. The answer requires current private documents

E. The model needs clearer task instructions  

**Answer:** D, E  
**Rationale:** Prompting handles instruction, structure, and format issues.
