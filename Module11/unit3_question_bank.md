# Module 11: Optimize Generative AI Model Performance with Microsoft Foundry

## Unit 3 Question Bank: Ground your model with Retrieval Augmented Generation

These questions are authored from the unit's Microsoft Learn concepts and rewritten as exam-style practice items. Answer choices are intentionally paraphrased instead of copied from the lesson text.

---

### 1. A chatbot must answer from current company policy documents. Which optimization strategy fits?

A. grounding data  
B. retriever  
C. grounded response  
D. Retrieval augmented generation (RAG)  

**Answer:** D  
**Rationale:** The requirement matches the unit capability named in the correct option.

---

### 2. What is the main purpose of "retrieval augmented generation"?

A. the component that finds relevant documents or passages  
B. an answer based on retrieved information rather than model memory alone  
C. retrieving relevant source content and adding it to the prompt for grounded generation  
D. trusted context provided to help the model answer accurately  

**Answer:** C  
**Rationale:** "retrieval augmented generation" is a core idea in this unit, and the correct option paraphrases its role.

---

### 3. A system searches a knowledge base before generating an answer. What component performs the search?

A. Object detection  
B. Retriever capability  
C. retrieval augmented generation  
D. Prompt engineering  

**Answer:** B  
**Rationale:** The correct option is the unit concept that best fits the stated requirement.

---

### 4. How does RAG differ from fine-tuning?

A. RAG supplies external context at request time; fine-tuning changes model behavior through additional training.  
B. RAG always changes model weights.  
C. Fine-tuning only searches documents.  
D. Both strategies are only for speech synthesis.  

**Answer:** A  
**Rationale:** The unit compares optimization strategies by how they improve responses.

### 5. What should a learner remember about "grounding data"?

A. a configuration task for the interface rather than the AI behavior  
B. a rule-based process that does not use model reasoning  
C. an access-control setting rather than the concept being defined  
D. grounding context provided to help the model answer accurately  

**Answer:** D  
**Rationale:** "grounding data" is tested by recognizing its purpose, not by memorizing wording from the page.

### 6. Why can RAG reduce unsupported answers?

A. RAG removes the need for source data.  
B. RAG is only a UI design pattern.  
C. The model receives relevant trusted content to use when generating the response.  
D. RAG prevents all prompts.  

**Answer:** C  
**Rationale:** Grounding helps responses stay tied to known information.

---

### 7. Which two components are part of a RAG pattern? Choose two.

A. A required audio microphone  
B. A generated image style  
C. No document content  
D. A retrieval source  
E. A prompt that includes retrieved context  

**Answer:** D, E  
**Rationale:** RAG combines retrieval with generation.

### 8. A model includes relevant policy excerpts in the prompt before answering. What are those excerpts?

**Answer:** A

### 9. Which three scenarios are good fits for RAG? Choose three.

A. Changing model weights for tone only  
B. Generating a voice waveform  
C. Detecting image edges  
D. Answers from product documentation  
E. Responses based on policy files  
F. Knowledge-base question answering  

**Answer:** D, E, F  
**Rationale:** RAG is useful when answers should be grounded in specific sources.

### 10. Which description best matches "retriever"?

B. Use a visual analysis capability when the input is only text.  
C. component that finds relevant documents or passages  
D. Choose a model based on popularity instead of task fit.  

**Answer:** C  
**Rationale:** The correct answer paraphrases how the unit explains "retriever".
