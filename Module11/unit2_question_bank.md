# Module 11: Optimize Generative AI Model Performance with Microsoft Foundry

## Unit 2 Question Bank: Optimize model output with prompt engineering

These questions are authored from the unit's Microsoft Learn concepts and rewritten as exam-style practice items. Answer choices are intentionally paraphrased instead of copied from the lesson text.

---

### 1. A model gives long unfocused answers, so a developer asks for exactly three bullets. What technique is being used?

A. clear instructions  
B. few-shot examples  
C. output format  
D. Prompt engineering  

**Answer:** D  
**Rationale:** The requirement matches the unit capability named in the correct option.

---

### 2. What is the main purpose of "prompt engineering"?

A. sample inputs and outputs that demonstrate the desired pattern  
B. the requested structure, such as bullets, JSON, or a table  
C. designing prompts to improve the usefulness, structure, and relevance of model output  
D. specific task guidance that reduces ambiguity  

**Answer:** C  
**Rationale:** "prompt engineering" is a core idea in this unit, and the correct option paraphrases its role.

---

### 3. A prompt includes two sample classifications before asking the model to classify a new case. What is this?

A. Information extraction  
B. Few-shot prompting  
C. Object detection  
D. Speech synthesis  

**Answer:** B  
**Rationale:** The correct option is the unit concept that best fits the stated requirement.

---

### 4. Why try prompt engineering before fine-tuning for many issues?

A. Prompt changes are faster and cheaper when the issue is instruction or format related.  
B. Prompt engineering requires retraining model weights.  
C. Fine-tuning is always required for every wording change.  
D. Prompts cannot influence output format.  

**Answer:** A  
**Rationale:** The unit presents prompt engineering as a practical first optimization strategy.

---

### 5. What should a learner remember about "clear instructions"?

A. a configuration task for the interface rather than the AI behavior  
B. a rule-based process that does not use model reasoning  
C. an access-control setting rather than the concept being defined  
D. explicit task instructions that reduces ambiguity  

**Answer:** D  
**Rationale:** "clear instructions" is tested by recognizing its purpose, not by memorizing wording from the page.

---

### 6. Why are examples useful in a prompt?

A. Examples are only used for image pixels.  
B. Examples remove the need to describe the task.  
C. They show the model the desired pattern for similar outputs.  
D. Examples prevent the model from answering.  

**Answer:** C  
**Rationale:** Few-shot examples guide response style and structure.

---

### 7. Which two prompt changes can improve output quality? Choose two.

A. Make instructions contradictory  
B. Omit the task  
C. Ask the model to ignore all context  
D. Specify the audience  
E. Define the response format  

**Answer:** D, E  
**Rationale:** Audience and format help produce more useful responses.

---

### 8. An app needs model output as JSON so code can parse it. What should the prompt specify?

A. Output format capability  
B. Model evaluation  
C. Choose a model based on popularity instead of task fit.  
D. Rely on one successful trial instead of broader evaluation.  

**Answer:** A  
**Rationale:** The scenario points to the unit capability described by the correct option.

---

### 9. Which three are prompt engineering techniques? Choose three.

A. Changing model weights  
B. Indexing private files only  
C. Sampling microphone input  
D. Clear task instructions  
E. Few-shot examples capability  
F. Output constraints  

**Answer:** D, E, F  
**Rationale:** Prompt engineering works through instructions, examples, and constraints.

### 10. Which description best matches "few-shot examples"?

A. Handle sensitive data without the required access controls.  
B. Leave the model without clear task instructions.  
C. few-shot examples that demonstrate the desired pattern  
D. Use a visual analysis capability when the input is only text.  

**Answer:** C  
**Rationale:** The correct answer paraphrases how the unit explains "few-shot examples".
