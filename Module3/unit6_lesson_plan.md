# Module 3: Introduction to Natural Language Processing Concepts

## Unit 6: Module Assessment Prep

### Source

Microsoft Learn: [Module assessment](https://learn.microsoft.com/en-us/training/modules/introduction-language/5-knowledge-check)

### Unit Overview

This unit prepares learners for the Module 3 assessment without copying Microsoft Learn assessment items. Learners review tokenization, preprocessing, statistical analysis, TF-IDF, bag-of-words, TextRank, embeddings, and semantic models.

### Learning Objectives

By the end of this unit, learners should be able to:

- Review the major NLP concepts in Module 3.
- Distinguish statistical techniques from semantic models.
- Match NLP techniques to scenarios.
- Explain key terms in beginner-friendly language.

### Key Terms

| Term | Definition |
| --- | --- |
| Statistical analysis | Techniques based on counts, frequencies, and probabilities. |
| Semantic analysis | Techniques that represent and compare meaning. |
| Assessment prep | Original review practice before a module assessment. |

### Lesson Flow

#### 1. Review Tokenization

Review corpus, token, preprocessing, n-grams, stemming, lemmatization, and POS tagging.

#### 2. Review Statistical Techniques

Review frequency analysis, TF-IDF, bag-of-words, Naive Bayes, and TextRank.

#### 3. Review Semantic Models

Review embeddings, vector similarity, attention, and semantic applications.

### Suggested Activities

#### Activity 1: Concept Sorting

Learners sort terms into tokenization, statistical, or semantic categories.

#### Activity 2: Scenario Match

Learners choose the best NLP technique for short scenarios.

### Checks for Understanding

1. What is tokenization?
2. Why is TF-IDF useful?
3. What are embeddings?
4. How do statistical and semantic techniques differ?
