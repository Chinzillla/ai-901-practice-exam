# Module 3: Introduction to Natural Language Processing Concepts

## Unit 4: Semantic Language Models

### Source

Microsoft Learn: [Semantic language models](https://learn.microsoft.com/en-us/training/modules/introduction-language/4-semantic-models)

### Unit Overview

This unit explains semantic language models that represent language tokens as vector embeddings. Learners explore semantic similarity, cosine similarity, vector arithmetic, analogy reasoning, and how semantic models support summarization, keyword extraction, entity recognition, and classification.

### Learning Objectives

By the end of this unit, learners should be able to:

- Describe embeddings as vector representations of language meaning.
- Explain why semantically similar words have similar vector orientations.
- Define cosine similarity at a conceptual level.
- Explain vector arithmetic examples such as dog plus young producing puppy-like meaning.
- Identify text analysis tasks supported by semantic models.

### Key Terms

| Term | Definition |
| --- | --- |
| Semantic model | A model that captures meaning and relationships between language tokens. |
| Embedding | A vector representation of a token or sentence. |
| Vector | A numeric representation with multiple dimensions. |
| Cosine similarity | A calculation used to compare vector orientation and semantic similarity. |
| Contextualized embedding | An embedding that reflects the surrounding context of a token. |
| Word2Vec | An earlier vector-based language modeling technique. |
| GloVe | An earlier vector-based language modeling technique. |

### Lesson Flow

#### 1. From Statistics to Meaning

Explain that semantic models go beyond counting words by encoding meaning in vectors.

#### 2. Embeddings and Vector Space

Describe embeddings as points or directions in multidimensional space. Similar meanings tend to have similar orientations.

#### 3. Measuring Similarity

Explain cosine similarity conceptually: it compares vector directions to estimate semantic relatedness.

#### 4. Vector Arithmetic

Use intuitive examples such as dog plus young being close to puppy, or puppy minus young being close to dog.

#### 5. Contextualized Embeddings and Attention

Explain that attention can consider surrounding tokens, producing embeddings that reflect context.

#### 6. Text Analysis Applications

Review summarization, keyword extraction, named entity recognition, and classification.

### Suggested Activities

#### Activity 1: Related Terms

Learners group words by semantic similarity.

#### Activity 2: Odd One Out

Learners identify which word is least related and explain why.

#### Activity 3: Semantic Task Match

Learners match semantic models to text analysis use cases.

### Checks for Understanding
