# Module 3: Introduction to Natural Language Processing Concepts

## Unit 7 Question Bank: Summary

These questions provide cumulative review for Module 3.

---

### 1. What is tokenization?

A. Breaking text into smaller units  
B. Generating speech audio  
C. Detecting objects in video  
D. Encrypting files  

**Answer:** A  
**Rationale:** Tokenization divides text into tokens for analysis.

---

### 2. What does frequency analysis help identify?

A. Common terms and possible topics  
B. Camera resolution  
C. Audio volume only  
D. Network traffic routes  

**Answer:** A  
**Rationale:** Frequent terms can reveal themes in text.

---

### 3. Which technique helps distinguish terms across documents?

A. TF-IDF  
B. Text-to-speech  
C. Object detection  
D. Face verification  

**Answer:** A  
**Rationale:** TF-IDF highlights distinctive terms.

---

### 4. What are embeddings?

A. Vector representations of meaning  
B. Printed summaries  
C. Audio samples  
D. Hardware diagrams  

**Answer:** A  
**Rationale:** Embeddings encode semantic relationships in numeric vectors.

---

### 5. Which task assigns documents to categories?

A. Text classification  
B. Speech synthesis  
C. Computer vision  
D. Image generation  

**Answer:** A  
**Rationale:** Text classification categorizes text documents.

---

### 6. True or False: Semantic language models can help group related words.

**Answer:** True  
**Rationale:** Similar meanings can produce similar vector representations.

---

### 7. True or False: Statistical text analysis and semantic models are exactly the same.

**Answer:** False  
**Rationale:** Statistical techniques rely heavily on counts and probabilities; semantic models represent meaning with vectors.

---

### 8. True or False: NLP can support summarization.

**Answer:** True  
**Rationale:** Summarization is a common NLP task.
