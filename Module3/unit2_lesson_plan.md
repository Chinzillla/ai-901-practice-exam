# Module 3: Introduction to Natural Language Processing Concepts

## Unit 2: Tokenization

### Source

Microsoft Learn: [Tokenization](https://learn.microsoft.com/en-us/training/modules/introduction-language/2-how-it-works)

### Unit Overview

This unit explains tokenization as the first step in many text analysis workflows. Learners explore tokens, corpora, frequency counting, and common preprocessing techniques such as normalization, stop word removal, n-grams, stemming, lemmatization, and parts-of-speech tagging.

### Learning Objectives

By the end of this unit, learners should be able to:

- Define corpus, token, and tokenization.
- Explain why tokens are useful for text analysis.
- Describe common preprocessing techniques.
- Distinguish between stemming and lemmatization.
- Recognize when n-grams and POS tagging are useful.

### Key Terms

| Term | Definition |
| --- | --- |
| Corpus | A body or collection of text used for analysis. |
| Token | A smaller unit of text, such as a word, subword, phrase, or punctuation sequence. |
| Tokenization | Breaking text into tokens for analysis. |
| Text normalization | Standardizing text, often by lowercasing or removing punctuation. |
| Stop word removal | Excluding low-meaning common words such as "the" or "a". |
| N-gram | A sequence of one or more terms, such as unigram, bigram, or trigram. |
| Stemming | Reducing words by removing endings. |
| Lemmatization | Reducing words to valid dictionary forms. |
| POS tagging | Labeling tokens with grammatical roles such as noun or verb. |

### Lesson Flow

#### 1. Corpus and Tokens

Explain that a corpus is a body of text and tokenization breaks that text into smaller units for analysis.

#### 2. Frequency and Meaning

Explain that counting token frequency can help identify important terms or themes.

#### 3. Preprocessing Choices

Review normalization, stop word removal, n-grams, stemming, lemmatization, and POS tagging.

#### 4. Tradeoffs

Explain that preprocessing can improve analysis, but it can also remove useful meaning. For example, lowercasing might lose the distinction between a person's name and a common word.

### Suggested Activities

#### Activity 1: Tokenize a Phrase

Learners break a short sentence into tokens and count repeated terms.

#### Activity 2: Preprocessing Debate

Learners decide whether to remove punctuation, remove stop words, or preserve case for different scenarios.
