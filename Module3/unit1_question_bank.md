# Module 3: Introduction to Natural Language Processing Concepts

## Unit 1 Question Bank: Introduction

These questions reinforce NLP and text analysis fundamentals.

---

### 1. What is text analysis?

A. NLP techniques that extract meaning, structure, or insights from text  
B. A method for replacing computer hardware  
C. A tool for formatting hard drives  
D. A way to compress video files only  

**Answer:** A  
**Rationale:** Text analysis uses NLP to work with natural language text.

---

### 2. Which source is an example of unstructured text?

A. Customer feedback comments  
B. A CPU temperature sensor only  
C. A network cable label only  
D. A blank spreadsheet  

**Answer:** A  
**Rationale:** Customer feedback is written language that often needs analysis to extract meaning.

---

### 3. Which task identifies the language used in a document?

A. Language detection  
B. Speech synthesis  
C. Image classification  
D. Object detection  

**Answer:** A  
**Rationale:** Language detection determines which language or languages are present.

---

### 4. Which task identifies important words or phrases in text?

A. Key term extraction  
B. Text-to-speech  
C. Semantic segmentation  
D. Facial recognition  

**Answer:** A  
**Rationale:** Key term extraction finds important topics or phrases.

---

### 5. Which task finds names of people, places, dates, or organizations?

A. Entity detection  
B. Image generation  
C. Speech translation  
D. File compression  

**Answer:** A  
**Rationale:** Entity detection finds named entities in text.

---

### 6. Which task identifies sensitive personal details?

A. PII detection  
B. Object detection  
C. Speech synthesis  
D. Prompt completion  

**Answer:** A  
**Rationale:** PII detection identifies personal information such as names, phone numbers, and account details.

---

### 7. Which task categorizes text as spam or not spam?

A. Text classification  
B. Computer vision  
C. Speech recognition  
D. OCR only  

**Answer:** A  
**Rationale:** Text classification assigns text to categories.

---

### 8. Which task predicts whether a social media post is positive, neutral, or negative?

A. Sentiment analysis  
B. Form extraction  
C. Speech synthesis  
D. Image classification  

**Answer:** A  
**Rationale:** Sentiment analysis classifies opinion or emotional tone.

---

### 9. Which task reduces a long document while keeping key points?

A. Summarization  
B. Object detection  
C. Text-to-speech only  
D. Data encryption  

**Answer:** A  
**Rationale:** Summarization condenses text while preserving important ideas.

---

### 10. Why is text analysis challenging?

A. Human language is complex and meaning depends on context  
B. Text always has only one meaning  
C. Computers naturally understand all language without models  
D. Text analysis never uses AI  

**Answer:** A  
**Rationale:** Ambiguity, context, and semantic meaning make text hard to analyze.

---

### 11. True or False: Text analysis can help turn support tickets into actionable insights.

**Answer:** True  
**Rationale:** Text analysis can classify, summarize, and extract entities from support tickets.

---

### 12. True or False: Sentiment analysis is unrelated to text classification.

**Answer:** False  
**Rationale:** Sentiment analysis is a specialized form of text classification.
