# Module 3: Introduction to Natural Language Processing Concepts

## Unit 5: Exercise - Explore Text Analytics

### Source

Microsoft Learn: [Exercise - Explore text analytics](https://learn.microsoft.com/en-us/training/modules/introduction-language/4b-exercise)

### Unit Overview

This exercise unit gives learners an opportunity to explore AI text analytics tasks. Learners should connect tool behavior to concepts such as tokenization, entity detection, key term extraction, sentiment analysis, summarization, and semantic models.

### Learning Objectives

By the end of this unit, learners should be able to:

- Observe common text analytics capabilities.
- Match observed features to NLP concepts.
- Reflect on which techniques support different text analysis outcomes.

### Key Terms

| Term | Definition |
| --- | --- |
| Text analytics | AI capabilities for analyzing text and extracting insights. |
| Feature mapping | Connecting an observed tool feature to the AI capability that supports it. |

### Lesson Flow

#### 1. Exercise Orientation

Explain that learners should actively observe which NLP capabilities are demonstrated.

#### 2. Capability Mapping

Map observed behaviors to NLP tasks such as key term extraction, entity detection, sentiment analysis, or summarization.

#### 3. Debrief

Discuss which capabilities were easiest to identify and which require more review.

### Suggested Activities

#### Activity 1: Capability Checklist

Learners mark each NLP capability they observe.

#### Activity 2: Evidence Notes

Learners record one example input and one observed output.

### Checks for Understanding

1. Which NLP capabilities did you observe?
2. Which output showed entity detection?
3. Which output showed sentiment analysis?
4. Which task was most useful for summarizing information?
