# Module 3: Introduction to Natural Language Processing Concepts

## Unit 6 Question Bank: Module Assessment Prep

These original questions prepare learners for Module 3 assessment concepts.

---

### 1. Which concept breaks a corpus into smaller units?

A. Tokenization  
B. Speech synthesis  
C. Object detection  
D. Image generation  

**Answer:** A  
**Rationale:** Tokenization breaks text into tokens.

---

### 2. Which technique removes common low-value words from analysis?

A. Stop word removal  
B. Object detection  
C. Text-to-speech  
D. Field extraction  

**Answer:** A  
**Rationale:** Stop word removal excludes common words such as "the" or "a."

---

### 3. Which technique highlights terms important in one document but rare across others?

A. TF-IDF  
B. Speech recognition  
C. Semantic segmentation  
D. Text-to-speech  

**Answer:** A  
**Rationale:** TF-IDF identifies distinctive document terms.

---

### 4. Which model representation ignores grammar and word order?

A. Bag-of-words  
B. Contextual embeddings  
C. Speech synthesis  
D. OCR  

**Answer:** A  
**Rationale:** Bag-of-words represents text using term counts or occurrences.

---

### 5. Which concept represents meaning as vectors?

A. Embeddings  
B. Stop words  
C. Punctuation only  
D. Disk paths  

**Answer:** A  
**Rationale:** Embeddings encode token or text meaning in vector form.

---

### 6. Which calculation compares vector orientations?

A. Cosine similarity  
B. Audio sampling  
C. Image cropping  
D. File hashing only  

**Answer:** A  
**Rationale:** Cosine similarity estimates semantic closeness by comparing vector directions.

---

### 7. Which method can select important original sentences for a summary?

A. TextRank  
B. Speech translation  
C. Object detection  
D. Disk encryption  

**Answer:** A  
**Rationale:** TextRank can support extractive summarization.

---

### 8. True or False: Lemmatization aims to produce valid base words.

**Answer:** True  
**Rationale:** Lemmatization uses linguistic rules and vocabulary to return valid forms.

---

### 9. True or False: Semantic models are based only on raw word counts.

**Answer:** False  
**Rationale:** Semantic models represent meaning with vectors and relationships.

---

### 10. True or False: NLP techniques can support classification, summarization, and entity recognition.

**Answer:** True  
**Rationale:** These are common NLP text analysis tasks.
