# Module 3: Introduction to Natural Language Processing Concepts

## Unit 5 Question Bank: Exercise - Explore Text Analytics

These questions reinforce the exercise goal of identifying text analytics capabilities.

---

### 1. What should learners focus on during the text analytics exercise?

A. Mapping observed features to NLP capabilities  
B. Ignoring tool outputs  
C. Replacing all text with images  
D. Configuring hardware  

**Answer:** A  
**Rationale:** The exercise reinforces concept recognition through observed behavior.

---

### 2. Which capability identifies named people, places, dates, or organizations?

A. Entity detection  
B. Speech synthesis  
C. Object detection  
D. Image generation  

**Answer:** A  
**Rationale:** Entity detection finds named items in text.

---

### 3. Which capability identifies important topics or phrases?

A. Key term extraction  
B. Text-to-speech  
C. Facial recognition  
D. Semantic segmentation  

**Answer:** A  
**Rationale:** Key term extraction identifies important words or phrases.

---

### 4. Which capability evaluates opinion as positive, neutral, or negative?

A. Sentiment analysis  
B. OCR only  
C. Speech translation  
D. Object detection  

**Answer:** A  
**Rationale:** Sentiment analysis classifies emotional or opinion tone.

---

### 5. Which capability reduces text while retaining main points?

A. Summarization  
B. Disk formatting  
C. Facial recognition  
D. Audio synthesis only  

**Answer:** A  
**Rationale:** Summarization condenses text while keeping key ideas.

---

### 6. True or False: Exercise units should connect tool behavior to concepts.

**Answer:** True  
**Rationale:** Observing outputs is useful only if learners connect them to the underlying capability.

---

### 7. True or False: Text analytics is only useful for perfectly structured tables.

**Answer:** False  
**Rationale:** Text analytics is useful for unstructured text such as reviews, tickets, and documents.

---

### 8. True or False: Sentiment analysis is a text analytics capability.

**Answer:** True  
**Rationale:** It analyzes text to classify sentiment.
