# Module 3: Introduction to Natural Language Processing Concepts

## Unit 2 Question Bank: Tokenization

These questions are authored from the unit's Microsoft Learn concepts and rewritten as exam-style practice items. Answer choices are intentionally paraphrased instead of copied from the lesson text.

### 1. An NLP pipeline splits a sentence into words and punctuation before analysis. Which step is this?

A. normalization  
B. stop words  
C. Prompt engineering  
D. Tokenization  

**Answer:** D  
**Rationale:** The requirement matches the unit capability named in the correct option.

---

### 2. What is the main purpose of "tokenization"?

A. standardizing text so equivalent words or forms can be analyzed consistently  
B. very common words that may be removed when they add little analytical value  
C. breaking text into smaller units that can be processed by language systems  
D. a word, word part, punctuation mark, or other text unit used for analysis  

**Answer:** C  
**Rationale:** "tokenization" is a core idea in this unit, and the correct option paraphrases its role.

### 3. A text analysis workflow treats 'Run,' 'run,' and 'running' more consistently. Which idea is involved?

A. Speech synthesis  
B. Text normalization  
C. token  
D. Object detection  

**Answer:** B  
**Rationale:** The correct option is the unit concept that best fits the stated requirement.

### 4. How does tokenization differ from semantic understanding?

A. Tokenization prepares text units; semantic understanding interprets meaning and relationships.  
B. Tokenization creates audio output, while semantics scans images.  
C. They are identical names for object detection.  
D. Semantic understanding only counts punctuation.  

**Answer:** A  
**Rationale:** Tokenization is an early preparation step, while semantic models deal with meaning.

### 5. What should a learner remember about "token"?

A. a configuration task for the interface rather than the AI behavior  
B. a rule-based process that does not use model reasoning  
C. an access-control setting rather than the concept being defined  
D. a token-sized text unit used for analysis  

**Answer:** D  
**Rationale:** "token" is tested by recognizing its purpose, not by memorizing wording from the page.

### 6. Why is tokenization important before statistical text analysis?

A. It replaces the need for language models.  
B. It converts text into a camera image.  
C. It creates the units that can be counted, compared, or modeled.  
D. It guarantees sentiment is always positive.  

**Answer:** C  
**Rationale:** Statistical NLP techniques depend on well-defined tokens.

### 7. Which two actions can be part of preparing text for NLP analysis? Choose two.

A. Generating a synthetic voice  
B. Detecting objects in an image  
C. Deploying a model endpoint  
D. Splitting text into tokens  
E. Normalizing words or casing  

**Answer:** D, E  
**Rationale:** Tokenization and normalization prepare text for analysis.

### 9. Which three items can affect token-based text analysis? Choose three.
