# Module 12: Implement a Responsible Generative AI Solution in Microsoft Foundry

## Unit 2 Question Bank: Plan a responsible generative AI solution

These questions are authored from the unit's Microsoft Learn concepts and rewritten as exam-style practice items. Answer choices are intentionally paraphrased instead of copied from the lesson text.

---

### 1. A team defines who will use an AI assistant and what tasks it should not handle. What planning concept is this?

A. responsible AI planning  
B. stakeholders  
C. risk mitigation  
D. Intended use and scope  

**Answer:** D  
**Rationale:** The requirement matches the unit capability named in the correct option.

---

### 2. What is the main purpose of "responsible AI planning"?

A. people affected by or responsible for the AI system  
B. planned controls that reduce possible harms from the solution  
C. identifying intended use, users, risks, and controls before building  
D. the approved purpose and context for the AI solution  

**Answer:** C  
**Rationale:** "responsible AI planning" is a core idea in this unit, and the correct option paraphrases its role.

---

### 3. A product owner, legal reviewer, developer, and support lead all review potential AI impacts. What group are they part of?

A. Object detection  
B. Stakeholders capability  
C. intended use  
D. Prompt engineering  

**Answer:** B  
**Rationale:** The correct option is the unit concept that best fits the stated requirement.

---

### 4. Why should responsible AI planning happen before implementation?

A. Early planning helps identify risks, users, constraints, and controls before design choices are locked in.  
B. Planning only matters after users report harm.  
C. Planning replaces all testing.  
D. Planning is unrelated to generative AI.  

**Answer:** A  
**Rationale:** The unit frames responsibility as part of solution planning.

---

### 5. What should a learner remember about "intended use"?

A. a configuration task for the interface rather than the AI behavior  
B. a rule-based process that does not use model reasoning  
C. an access-control setting rather than the concept being defined  
D. approved purpose and context for the AI solution  

**Answer:** D  
**Rationale:** "intended use" is tested by recognizing its purpose, not by memorizing wording from the page.

---

### 6. What should a team clarify when defining intended use?

A. A deployment setting that does not address user impact.  
B. A rule that no stakeholder can review it.  
C. What the solution is for, who will use it, and where it should not be used.  
D. Only the response wording, without defining allowed use.  

**Answer:** C  
**Rationale:** Intended use sets boundaries for responsible design.

---

### 7. Which two activities belong in responsible AI planning? Choose two.

A. Ignore potential harms  
B. Remove all human ownership  
C. Publish secrets in prompts  
D. Identify affected users  
E. Define acceptable use boundaries  

**Answer:** D, E  
**Rationale:** Planning includes user impact and use boundaries.

---

### 8. A team decides to require human review for high-impact outputs before deployment. What kind of planning is this?

A. Risk mitigation capability  
B. Speech synthesis  
C. Information extraction  
D. Model evaluation  

**Answer:** A  
**Rationale:** The scenario points to the unit capability described by the correct option.

---

### 9. Which three items should be considered when planning a responsible solution? Choose three.

A. Random model names  
B. Monitor resolution  
C. Unrelated file colors  
D. Intended users  
E. Possible harms  
F. Mitigation controls  

**Answer:** D, E, F  
**Rationale:** Responsible planning connects users, risks, and controls.

### 10. Which description best matches "stakeholders"?

A. Rely on one successful trial instead of broader evaluation.  
B. Use a visual analysis capability when the input is only text.  
C. stakeholders affected by or responsible for the AI system  
D. Choose a model based on popularity instead of task fit.  

**Answer:** C  
**Rationale:** The correct answer paraphrases how the unit explains "stakeholders".
