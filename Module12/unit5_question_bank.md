# Module 12: Implement a Responsible Generative AI Solution in Microsoft Foundry

## Unit 5 Question Bank: Mitigate potential harms

These questions are authored from the unit's Microsoft Learn concepts and rewritten as exam-style practice items. Answer choices are intentionally paraphrased instead of copied from the lesson text.

---

### 1. A team adds content filters after measuring harmful responses. What are they doing?

A. model layer  
B. safety system layer  
C. user experience layer  
D. Applying mitigation controls  

**Answer:** D  
**Rationale:** The requirement matches the unit capability named in the correct option.

### 2. What is the main purpose of "mitigation"?

A. platform controls such as filters and safeguards that reduce harmful outputs  
B. interface choices, disclosures, and human review that guide safe use  
C. applying controls that reduce identified AI risks  
D. choosing or adapting the model to better fit the intended use  

**Answer:** C  
**Rationale:** "mitigation" is a core idea in this unit, and the correct option paraphrases its role.

### 3. A solution uses a smaller task-specific model instead of a broad model to reduce irrelevant outputs. Which layer is involved?

A. Speech synthesis  
B. Model-level mitigation  
C. Prompt engineering  
D. Object detection  

**Answer:** B  
**Rationale:** The correct option is the unit concept that best fits the stated requirement.

### 4. How is mitigation different from measurement?

A. Measurement evaluates harm; mitigation changes the solution to reduce harm.  
B. Mitigation only counts test cases.  
C. Measurement applies content filters.  
D. Both steps mean skipping retesting.  

**Answer:** A  
**Rationale:** The responsible AI workflow measures risk and then applies controls.

### 5. What should a learner remember about "model layer"?

A. a configuration task for the interface rather than the AI behavior  
B. a rule-based process that does not use model reasoning  
C. an access-control setting rather than the concept being defined  
D. selecting or adapting the model to better fit the intended use  

**Answer:** D  
**Rationale:** "model layer" is tested by recognizing its purpose, not by memorizing wording from the page.

### 6. Why should a solution be retested after mitigation?

A. Mitigations always work perfectly the first time.  
B. Retesting is only for image classification.  
C. Retesting verifies whether the controls actually reduced the measured harm.  
D. Retesting removes all need for controls.  

**Answer:** C  
**Rationale:** The unit describes mitigation as iterative with measurement.

### 7. Which two are mitigation examples? Choose two.

**Rationale:** Safety filters and grounding are concrete harm-reduction controls.
