# Module 12: Implement a Responsible Generative AI Solution in Microsoft Foundry

## Unit 6: Manage a responsible generative AI solution

### Source

[Microsoft Learn unit](https://learn.microsoft.com/en-us/training/modules/responsible-ai-studio/6-operate-responsibly)

### Unit Overview

This unit focuses on prerelease reviews, phased rollout, incident response, rollback, blocking capabilities, feedback, and telemetry.

### Learning Objectives

- identify prerelease review areas
- explain phased delivery
- describe incident and rollback planning
- recognize feedback and telemetry needs

### Key Terms

| Term | Beginner-Friendly Definition |
| --- | --- |
| Prerelease review | A review by relevant teams before release, such as legal, privacy, security, and accessibility. |
| Phased delivery | Releasing gradually to a restricted group before broader availability. |
| Incident response plan | A plan for responding to unexpected harmful or problematic behavior. |
| Rollback plan | Defined steps for reverting to a previous state. |
| Telemetry | Operational data used to understand usage, satisfaction, gaps, and issues. |

### Lesson Flow

#### 1. Before release

- Complete legal, privacy, security, and accessibility reviews as appropriate.
- Confirm documentation and risk plans are ready.

#### 2. Operate

- Release to a smaller group first, gather feedback, monitor telemetry, and prepare incident response and rollback.
- Implement ways to block harmful responses or abusive users when needed.

### Suggested Activities

- Ask learners to summarize the unit in one sentence using beginner-friendly language.
- Give learners a short workplace scenario and have them choose the correct concept, tool, strategy, or responsibility.
- Have learners identify one exam clue word from the unit, such as current, uploaded, benchmark, grounded, fine-tuned, guardrail, or endpoint.
- Pair learners to explain why one tempting alternative is not the best answer.

### Checks for Understanding

- What problem does this unit's main concept solve?
- Which keyword in a scenario would point you toward this concept?
- What is one limitation or caution learners should remember?
- How would this concept appear in a simple AI-901-style business scenario?

### Assessment

Use the accompanying question bank to check recognition, scenario selection, vocabulary, and common misconceptions. Learners should be able to explain not only the correct answer, but why the distractors are less appropriate.

### Teaching Notes

- Keep the explanation practical and scenario-based.
- Emphasize the difference between similar concepts instead of asking learners to memorize product names only.
- Connect the unit to AI-901 readiness by asking, "What would the exam scenario be asking me to choose?"
- Reinforce that secure, responsible, and measured development choices matter in generative AI apps.

### Estimated Duration

45-60 minutes
