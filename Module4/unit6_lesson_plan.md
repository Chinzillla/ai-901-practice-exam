# Module 4: Introduction to AI Speech Concepts

## Unit 6: Module Assessment Prep

### Source

Microsoft Learn: [Module assessment](https://learn.microsoft.com/en-us/training/modules/introduction-ai-speech/6-knowledge-check)

### Unit Overview

This unit prepares learners for the Module 4 assessment without copying Microsoft Learn assessment items. Learners review speech recognition, speech synthesis, pipeline stages, phonemes, feature extraction, language modeling, prosody, and audio generation.

### Learning Objectives

By the end of this unit, learners should be able to:

- Review recognition and synthesis concepts.
- Distinguish speech-to-text from text-to-speech.
- Identify key recognition pipeline stages.
- Identify key synthesis pipeline stages.
- Apply speech concepts to scenario questions.

### Key Terms

| Term | Definition |
| --- | --- |
| Assessment prep | Original review before a module assessment. |
| Recognition pipeline | The stages that convert audio to text. |
| Synthesis pipeline | The stages that convert text to audio. |

### Lesson Flow

#### 1. Review Speech Recognition

Review audio capture, preprocessing, acoustic modeling, language modeling, decoding, and post-processing.

#### 2. Review Speech Synthesis

Review text normalization, linguistic analysis, prosody generation, and audio synthesis.

#### 3. Scenario Practice

Use scenario questions that ask learners to identify the correct stage or capability.

### Suggested Activities

#### Activity 1: Pipeline Compare

Learners compare the recognition and synthesis pipelines side by side.

#### Activity 2: Stage Match

Learners match stage names to descriptions.

### Checks for Understanding

1. What does recognition convert?
2. What does synthesis convert?
3. What are phonemes?
4. What is prosody?
