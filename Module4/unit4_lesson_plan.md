# Module 4: Introduction to AI Speech Concepts

## Unit 4: Speech Synthesis

### Source

Microsoft Learn: [Speech synthesis](https://learn.microsoft.com/en-us/training/modules/introduction-ai-speech/4-speech-synthesis)

### Unit Overview

This unit explains speech synthesis, also called text-to-speech, which converts written text into spoken audio. Learners explore text normalization, linguistic analysis, grapheme-to-phoneme conversion, prosody generation, and waveform generation.

### Learning Objectives

By the end of this unit, learners should be able to:

- Define speech synthesis and text-to-speech.
- Identify the major stages in the synthesis pipeline.
- Explain why text normalization is needed.
- Define graphemes, phonemes, and G2P conversion.
- Explain prosody and why it affects naturalness.
- Recognize the role of neural vocoders.

### Key Terms

| Term | Definition |
| --- | --- |
| Speech synthesis | Converting written text into spoken audio. |
| Text-to-speech (TTS) | Another name for speech synthesis. |
| Text normalization | Expanding or standardizing text for pronunciation. |
| Grapheme | A written character or letter pattern. |
| Phoneme | A speech sound unit. |
| G2P | Grapheme-to-phoneme conversion. |
| Prosody | Rhythm, stress, pitch, pauses, and intonation in speech. |
| Neural vocoder | A model that generates audio waveforms from acoustic representations. |

### Lesson Flow

#### 1. Text Normalization

Explain that raw text must be expanded into speakable form. Examples include abbreviations, numbers, dates, times, and symbols.

#### 2. Linguistic Analysis

Explain that the system maps text to phonemes and determines pronunciation.

#### 3. Grapheme-to-Phoneme Conversion

Explain G2P as mapping written characters to speech sounds, especially important when spelling does not clearly indicate pronunciation.

#### 4. Prosody Generation

Explain that prosody determines how speech sounds: rhythm, stress, pitch, pauses, duration, and intensity.

#### 5. Audio Generation

Explain that neural vocoders generate final waveforms from phonemes and prosody specifications.

### Suggested Activities

#### Activity 1: Normalize the Text

Learners convert abbreviations, numbers, dates, and currency into spoken forms.

#### Activity 2: Prosody Practice

Learners read the same sentence with different emphasis and discuss how meaning changes.
