# Module 4: Introduction to AI Speech Concepts

## Unit 6 Question Bank: Module Assessment Prep

These original questions prepare learners for Module 4 assessment concepts.

---

### 1. Which pipeline converts audio into text?

A. Speech recognition  
B. Speech synthesis  
C. Computer vision  
D. Information extraction only  

**Answer:** A  
**Rationale:** Speech recognition converts spoken audio into text.

---

### 2. Which pipeline converts text into audio?

A. Speech synthesis  
B. Speech recognition  
C. Object detection  
D. Entity detection  

**Answer:** A  
**Rationale:** Speech synthesis is text-to-speech.

---

### 3. Which stage extracts compact features from audio?

A. Preprocessing  
B. Prosody generation  
C. Text normalization  
D. Field mapping  

**Answer:** A  
**Rationale:** Preprocessing extracts features such as MFCCs from audio.

---

### 4. Which stage applies vocabulary and grammar to resolve ambiguity?

A. Language modeling  
B. Audio capture only  
C. Text normalization only  
D. Neural vocoding only  

**Answer:** A  
**Rationale:** Language models help choose plausible word sequences.

---

### 5. Which stage expands "Dr." into "Doctor" for pronunciation?

A. Text normalization  
B. Audio capture  
C. Object detection  
D. Beam search decoding  

**Answer:** A  
**Rationale:** Text normalization prepares written text for speech output.

---

### 6. Which concept describes rhythm, stress, pitch, and pauses?

A. Prosody  
B. OCR  
C. TF-IDF  
D. Object detection  

**Answer:** A  
**Rationale:** Prosody affects the naturalness and meaning of speech.

---

### 7. True or False: Phonemes are small units of sound in speech.

**Answer:** True  
**Rationale:** Phonemes distinguish words in spoken language.

---

### 8. True or False: Speech synthesis and speech recognition both start with written text.

**Answer:** False  
**Rationale:** Recognition starts with audio; synthesis starts with text.

---

### 9. True or False: Post-processing can improve transcription readability.

**Answer:** True  
**Rationale:** It can add capitalization, punctuation, and formatting.

---

### 10. True or False: Prosody helps speech sound natural.

**Answer:** True  
**Rationale:** Prosody controls rhythm, stress, and intonation.
