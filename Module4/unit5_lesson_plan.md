# Module 4: Introduction to AI Speech Concepts

## Unit 5: Exercise - Explore AI Speech

### Source

Microsoft Learn: [Exercise - Explore AI speech](https://learn.microsoft.com/en-us/training/modules/introduction-ai-speech/5-exercise-speech)

### Unit Overview

This exercise unit gives learners practice using speech-to-text and text-to-speech with an AI chatbot. The focus is on observing how speech recognition and synthesis work together in a conversational experience.

### Learning Objectives

By the end of this unit, learners should be able to:

- Identify speech-to-text in a chatbot experience.
- Identify text-to-speech in a chatbot experience.
- Explain how both capabilities support two-way voice interaction.
- Reflect on usability, accessibility, and accuracy considerations.

### Key Terms

| Term | Definition |
| --- | --- |
| Speech-to-text | Converting spoken audio into text. |
| Text-to-speech | Converting written text into spoken audio. |
| Voice chatbot | A chatbot that supports spoken input, spoken output, or both. |

### Lesson Flow

#### 1. Exercise Orientation

Explain that learners should observe both input and output sides of the voice interaction.

#### 2. Speech-to-Text Observation

Ask learners to notice how spoken input appears as text.

#### 3. Text-to-Speech Observation

Ask learners to notice how the bot's text response is converted into audio.

#### 4. Reflection

Discuss accuracy, noise, latency, and alternatives for accessibility.

### Suggested Activities

#### Activity 1: Observation Log

Learners record one spoken prompt, the transcribed text, and the spoken response.

#### Activity 2: Environment Test

Learners consider how noise or microphone quality might affect the experience.

### Checks for Understanding

1. Where did speech-to-text appear?
2. Where did text-to-speech appear?
3. What could reduce speech recognition accuracy?
4. Why should text alternatives remain available?
