# Module 5: Introduction to Computer Vision Concepts

## Unit 9 Question Bank: Summary

---

### 1. Computer vision is built on analysis of:

A. Numeric pixel values  
B. Spoken words only  
C. Calendar events  
D. Email headers only  

**Answer:** A  
**Rationale:** Digital images are represented by pixel values.

---

### 2. Which task predicts one main image label?

A. Image classification  
B. Speech recognition  
C. Field mapping  
D. Tokenization  

**Answer:** A  
**Rationale:** Classification predicts an image label.

---

### 3. Which task identifies object locations?

A. Object detection  
B. Sentiment analysis  
C. Speech synthesis  
D. Text normalization  

**Answer:** A  
**Rationale:** Detection locates objects.

---

### 4. Which architecture uses image patches and attention?

A. Vision transformer  
B. Bag-of-words  
C. Speech synthesizer  
D. TF-IDF  

**Answer:** A  
**Rationale:** Vision transformers use patches and attention.

---

### 5. True or False: Modern models can both interpret visual input and generate visual output.

**Answer:** True  
**Rationale:** Multimodal and generative models can support interpretation and generation.

---

### 6. True or False: Computer vision applies only to still photographs, not video.

**Answer:** False  
**Rationale:** Computer vision can process photos, videos, and live camera feeds.
