# Module 5: Introduction to Computer Vision Concepts

## Unit 4: Convolutional Neural Networks

### Source

Microsoft Learn: [Convolutional neural networks](https://learn.microsoft.com/en-us/training/modules/introduction-computer-vision/4-computer-vision-models)

### Unit Overview

This unit introduces convolutional neural networks (CNNs), a deep learning architecture that uses filters to extract image features and feed them into a neural network for predictions such as image classification.

### Learning Objectives

- Define convolutional neural network.
- Explain feature maps.
- Describe pooling and flattening at a high level.
- Explain how CNNs produce class probabilities.
- Recognize training concepts such as labels, loss, epochs, and learned weights.

### Key Terms

| Term | Definition |
| --- | --- |
| CNN | A neural network architecture that uses convolutional filters for image features. |
| Feature map | Numeric output produced by applying filters to an image. |
| Pooling | Reducing feature map size while preserving important features. |
| Flattening | Converting feature maps into a one-dimensional feature array. |
| Softmax | A function that can output class probabilities. |
| Loss | Difference between predicted and actual results during training. |
| Epoch | One pass through training data. |

### Lesson Flow

#### 1. Why CNNs Matter

Explain that computer vision often needs models that learn meaningful visual features from many labeled images.

#### 2. Filters and Feature Maps

Explain that CNN filters extract numeric feature maps from images.

#### 3. Classification Pipeline

Review filters, pooling, flattening, fully connected layers, and output probabilities.

#### 4. Training

Explain that training compares predictions to labels, calculates loss, and adjusts weights over many epochs.

### Suggested Activities

- Have learners order CNN processing steps.
- Ask learners to explain why labeled images are needed for classification training.

### Checks for Understanding
