# Module 5: Introduction to Computer Vision Concepts

## Unit 8 Question Bank: Module Assessment Prep

---

### 1. Which values are fundamental to digital image analysis?

A. Pixel values  
B. Spoken words only  
C. Invoice totals only  
D. Prompt history only  

**Answer:** A  
**Rationale:** Computer vision is based on numeric pixel data.

---

### 2. Which task locates objects in an image?

A. Object detection  
B. Speech synthesis  
C. Language detection  
D. Text classification  

**Answer:** A  
**Rationale:** Detection identifies object locations.

---

### 3. Which model architecture uses filters to extract image features?

A. CNN  
B. Speech recognizer  
C. Bag-of-words  
D. OCR postprocessor only  

**Answer:** A  
**Rationale:** CNNs use convolutional filters.

---

### 4. Which model uses attention over image patches?

A. Vision transformer  
B. Naive Bayes  
C. Speech synthesizer only  
D. TextRank  

**Answer:** A  
**Rationale:** ViTs use transformer attention with image patches.

---

### 5. Which model can connect text and images?

A. Multimodal model  
B. Grayscale filter only  
C. Stop word remover  
D. Audio sampler  

**Answer:** A  
**Rationale:** Multimodal models relate visual and language representations.

---

### 6. Which process can generate a new image from a prompt?

A. Image generation  
B. Speech recognition  
C. TF-IDF  
D. Field extraction only  

**Answer:** A  
**Rationale:** Image generation creates visual output.

---

### 7. True or False: Semantic segmentation works at pixel level.

**Answer:** True  
**Rationale:** It classifies individual pixels by object or region.

---

### 8. True or False: CNNs are unrelated to computer vision.

**Answer:** False  
**Rationale:** CNNs are a common computer vision architecture.
