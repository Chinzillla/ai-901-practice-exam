# Module 5: Introduction to Computer Vision Concepts

## Unit 7: Exercise - Explore Computer Vision

### Source

Microsoft Learn: [Exercise - Explore computer vision](https://learn.microsoft.com/en-us/training/modules/introduction-computer-vision/5b-exercise)

### Unit Overview

This exercise unit asks learners to explore a computer vision model that identifies image contents. Learners should focus on mapping observed model outputs to vision concepts.

### Learning Objectives

- Observe computer vision predictions.
- Identify whether outputs resemble classification, detection, or description.
- Reflect on model confidence and limitations.

### Key Terms

| Term | Definition |
| --- | --- |
| Model prediction | The output generated by a model for an input. |
| Confidence | A score or indication of how certain a model is. |

### Lesson Flow

#### 1. Exercise Orientation

Explain that learners should observe the image input and prediction output.

#### 2. Output Mapping

Have learners classify outputs as labels, object locations, tags, or descriptions.

#### 3. Debrief

Discuss what the model handled well and where uncertainty appeared.

### Suggested Activities

- Keep an observation log of image, output, and likely technique.
- Compare predictions for simple versus complex images.

### Checks for Understanding

1. What kind of output did the model provide?
2. Which vision concept does that output match?
3. Why might confidence matter?
