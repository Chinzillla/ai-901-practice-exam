# Module 5: Introduction to Computer Vision Concepts

## Unit 8: Module Assessment Prep

### Source

Microsoft Learn: [Module assessment](https://learn.microsoft.com/en-us/training/modules/introduction-computer-vision/6-knowledge-check)

### Unit Overview

This unit prepares learners for the Module 5 assessment without copying Microsoft Learn assessment items. It reviews pixels, filters, CNNs, vision transformers, multimodal models, and image generation.

### Learning Objectives

- Review key computer vision concepts.
- Distinguish classification, detection, segmentation, and generation.
- Explain pixels, filters, CNNs, and ViTs at a high level.
- Apply concepts to scenario questions.

### Key Terms

| Term | Definition |
| --- | --- |
| Assessment prep | Original review before a knowledge check. |
| Vision workload | A visual AI capability such as classification, detection, or generation. |

### Lesson Flow

#### 1. Review Tasks

Compare classification, detection, segmentation, contextual analysis, and generation.

#### 2. Review Model Foundations

Review pixels, filters, CNNs, ViTs, and multimodal models.

#### 3. Scenario Practice

Use short requirements to select the correct capability.

### Suggested Activities

- Learners create a comparison table of vision tasks.
- Learners answer rapid scenario questions.

### Checks for Understanding

1. What are pixels?
2. What do CNN filters extract?
3. What does a ViT process?
4. What does image generation create?
