# Module 5: Introduction to Computer Vision Concepts

## Unit 5: Vision Transformers and Multimodal Models

### Source

Microsoft Learn: [Vision transformers and multimodal models](https://learn.microsoft.com/en-us/training/modules/introduction-computer-vision/5-modern-vision-models)

### Unit Overview

This unit explains how computer vision has evolved from CNN-based models to vision transformers and multimodal models that connect visual features with language.

### Learning Objectives

- Describe how vision transformers process image patches.
- Explain attention in a visual context.
- Define multimodal model.
- Explain how language and vision embeddings can be combined.
- Recognize captioning and visual description scenarios.

### Key Terms

| Term | Definition |
| --- | --- |
| Vision transformer (ViT) | A transformer model that processes image patches. |
| Image patch | A region of pixel values used as input to a vision transformer. |
| Visual embedding | A vector representation of visual features. |
| Cross-model attention | A technique for relating embeddings from different modalities. |
| Multimodal model | A model that combines inputs such as text and images. |

### Lesson Flow

#### 1. From CNNs to Transformers

Explain that CNNs remain important, but transformers introduced powerful attention-based modeling.

#### 2. Vision Transformers

Explain that ViTs process patches of pixels instead of language tokens.

#### 3. Visual Relationships

Explain that attention can represent contextual relationships between image patches.

#### 4. Multimodal Models

Explain how combining language and visual embeddings allows models to describe images.

### Suggested Activities

- Compare language tokens to image patches.
- Ask learners to identify multimodal use cases.

### Checks for Understanding
