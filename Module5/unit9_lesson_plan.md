# Module 5: Introduction to Computer Vision Concepts

## Unit 9: Summary

### Source

Microsoft Learn: [Summary](https://learn.microsoft.com/en-us/training/modules/introduction-computer-vision/7-summary)

### Unit Overview

This unit summarizes computer vision concepts, including pixel values, image processing, CNNs, vision transformers, multimodal models, and visual generation.

### Learning Objectives

- Summarize how computer vision processes images.
- Review common vision tasks.
- Compare CNNs, ViTs, and multimodal models.
- Identify common vision scenarios.

### Key Terms

| Term | Definition |
| --- | --- |
| Pixel | Numeric image value. |
| CNN | Filter-based neural network for image features. |
| ViT | Transformer model for image patches. |
| Multimodal model | Model combining vision and language. |

### Lesson Flow

#### 1. Review Visual Data

Summarize pixels, channels, filters, and convolution.

#### 2. Review Vision Tasks

Summarize classification, detection, segmentation, description, and generation.

#### 3. Review Model Evolution

Summarize the progression from statistical classifiers to CNNs to transformer-based multimodal models.

### Suggested Activities

- Learners write one scenario for each vision task.
- Learners explain CNN versus ViT in one sentence each.

### Checks for Understanding

1. What is an image made of from a computer's perspective?
2. What does object detection do?
3. What does a multimodal model combine?
