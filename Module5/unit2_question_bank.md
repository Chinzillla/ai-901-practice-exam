### 1. A mobile app must decide whether a picture shows a cat, dog, or bird. Which task should it use?

A. object detection  
B. semantic segmentation  
C. optical character recognition  
D. Image classification  

**Answer:** D  
**Rationale:** This scenario matches Computer vision tasks and techniques as described in the unit.

### 2. What is the main purpose of "image classification"?

A. classifying pixels into categories or regions  
B. extracting readable text from images  
C. assigning one or more labels that describe an image  
D. identifying objects and their locations in an image  

**Answer:** C  
**Rationale:** "image classification" is a core idea in this unit, and the correct option paraphrases its role.

### 3. A security system must draw boxes around people and vehicles in camera images. Which task fits?

A. Information extraction  
B. Object detection for this scenario  
C. Prompt engineering  
D. Speech synthesis  

**Answer:** B  
**Rationale:** The correct option is the unit concept that best fits the stated requirement.

### 4. What is the key difference between object detection and semantic segmentation?

A. Object detection locates objects with regions; segmentation classifies pixels into areas.  
B. Detection reads text, while segmentation translates speech.  
C. Segmentation only creates new images from prompts.  
D. Both techniques only classify a whole image.  

**Answer:** A  
**Rationale:** The unit distinguishes locating objects from assigning pixel-level categories.

### 5. What should a learner remember about "object detection"?

D. identifying objects and their locations in an image for this scenario  

**Answer:** D  
**Rationale:** "object detection" is tested by recognizing its purpose, not by memorizing wording from the page.

### 6. Why does a vision task need the right technique selected up front?

A. Choosing a technique only changes the portal theme.
