# Module 5: Introduction to Computer Vision Concepts

## Unit 6: Image Generation

### Source

Microsoft Learn: [Image generation](https://learn.microsoft.com/en-us/training/modules/introduction-computer-vision/5a-generate-images)

### Unit Overview

This unit introduces image generation as a multimodal capability where natural language prompts can guide models to synthesize images or videos.

### Learning Objectives

- Explain image generation from prompts.
- Describe diffusion at a high level.
- Recognize that generation can start from random pixel noise.
- Explain how prompts guide visual features.
- Identify video generation as a related emerging scenario.

### Key Terms

| Term | Definition |
| --- | --- |
| Image generation | Creating images from prompts or other inputs. |
| Diffusion | An iterative generation process that removes noise to form an image. |
| Noise | Random pixel values used as a starting point in diffusion. |
| Prompt | Natural language description of the desired visual output. |

### Lesson Flow

#### 1. Natural Language to Visual Output

Explain that a prompt can describe an image, and a model can synthesize matching visual content.

#### 2. Diffusion Concept

Explain that many systems start with random pixels and iteratively remove noise while comparing progress to the prompt.

#### 3. Image and Video Generation

Explain that similar ideas can apply to video, with added challenges such as motion and temporal consistency.

### Suggested Activities

- Ask learners to write prompts that specify subject, setting, and style.
- Discuss why video generation requires temporal consistency.

### Checks for Understanding

1. What does image generation create?
2. What guides the generation process?
3. What is diffusion?
4. Why is video generation more complex?
