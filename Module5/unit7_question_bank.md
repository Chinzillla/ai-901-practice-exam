# Module 5: Introduction to Computer Vision Concepts

## Unit 7 Question Bank: Exercise - Explore Computer Vision

---

### 1. What should learners observe in a computer vision exercise?

A. The input image and model prediction  
B. Only network settings  
C. Only audio volume  
D. Only text prompts  

**Answer:** A  
**Rationale:** The exercise connects visual input to model output.

---

### 2. If a model outputs "apple" for an image, what task may it be performing?

A. Image classification  
B. Speech synthesis  
C. Tokenization  
D. Field extraction  

**Answer:** A  
**Rationale:** A single image label is classification-like output.

---

### 3. If a model marks object locations, what task may it be performing?

A. Object detection  
B. Sentiment analysis  
C. Text normalization  
D. RAG  

**Answer:** A  
**Rationale:** Marking locations is object detection.

---

### 4. Why might model confidence matter?

A. It helps users evaluate prediction certainty  
B. It converts images into speech  
C. It replaces all testing  
D. It removes visual input  

**Answer:** A  
**Rationale:** Confidence helps interpret prediction reliability.

---

### 5. True or False: Exercise observations should be mapped back to concepts.

**Answer:** True  
**Rationale:** Mapping outputs to concepts reinforces exam readiness.

---

### 6. True or False: Complex images may be harder for vision models than simple images.

**Answer:** True  
**Rationale:** More objects, clutter, or ambiguity can increase difficulty.
