# Module 5: Introduction to Computer Vision Concepts

## Unit 3: Images and Image Processing

### Source

Microsoft Learn: [Images and image processing](https://learn.microsoft.com/en-us/training/modules/introduction-computer-vision/3-understand-image-processing)

### Unit Overview

This unit explains how images are represented as numeric pixel values and how filters can transform images by convolving kernels across pixel arrays.

### Learning Objectives

- Explain pixels and resolution.
- Distinguish grayscale and RGB images.
- Describe channels.
- Explain filters, kernels, and convolution conceptually.
- Recognize edge detection as a filtering example.

### Key Terms

| Term | Definition |
| --- | --- |
| Pixel | A numeric value representing a point in an image. |
| Resolution | The number of pixel rows and columns in an image. |
| Channel | A layer of pixel values, such as red, green, or blue. |
| RGB | Red, green, and blue color channels. |
| Filter | A transformation applied to image pixel values. |
| Kernel | A small matrix of weights used by a filter. |
| Convolution | Sliding a kernel across an image to calculate transformed values. |

### Lesson Flow

#### 1. Images as Pixel Arrays

Explain that computers represent images with numeric pixel values.

#### 2. Grayscale and Color Images

Explain that grayscale can use one layer, while RGB images use three channels.

#### 3. Filters and Kernels

Describe how a filter kernel applies weighted calculations to image regions.

#### 4. Convolution and Edge Detection

Explain convolution as moving the kernel across the image and producing a transformed image, such as one with highlighted edges.

### Suggested Activities

- Draw a small grid of pixel values and apply a simple conceptual filter.
- Ask learners to identify channels in an RGB image.

### Checks for Understanding

1. What is a pixel?
2. What does resolution describe?
3. What are RGB channels?
4. What does convolution do?
