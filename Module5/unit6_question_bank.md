### 2. What is the main purpose of "image generation"?

A. visual direction such as realistic, diagrammatic, or illustrated  
B. refining prompts or outputs until the image better matches the goal

**Answer:** C  
**Rationale:** "image generation" is a core idea in this unit, and the correct option paraphrases its role.

### 3. A designer changes the wording to make an image more realistic and less cartoon-like. What is being refined?

**Rationale:** The correct option is the unit concept that best fits the stated requirement.

### 4. How is image generation different from image classification?

A. Generation creates new images; classification labels existing images.  
B. Generation only extracts text from receipts.  
C. Classification creates new pixels from prompts.  
D. Both are speech synthesis tasks.  

**Answer:** A  
**Rationale:** The unit separates creating visual content from analyzing existing images.

### 5. What should a learner remember about "prompt"?

### 6. Why are prompt details important for image generation?

A. Prompt details only affect audio volume.  
B. Prompts remove the need to review generated content.  
C. They guide subject, style, composition, and other visual properties of the output.  
D. Prompts are ignored by image models.  

**Answer:** C  
**Rationale:** Generated images depend heavily on prompt specificity.

### 7. Which two prompt details can guide an image-generation model? Choose two.

A. A speech sampling rate  
B. A database primary key  
C. A sentiment score  
D. The subject to show  
E. The desired visual style  

**Answer:** D, E  
**Rationale:** Subject and style are common prompt controls for image output.

### 9. Which three practices support better generated image results? Choose three.

A. Assume the first output is final  
B. Use only hidden system settings  
C. Ignore responsible AI review  
D. Describe the main subject

**Answer:** D, E, F  
**Rationale:** Clear prompting and iteration improve generated visual outputs.
