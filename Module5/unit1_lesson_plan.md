# Module 5: Introduction to Computer Vision Concepts

## Unit 1: Introduction

### Source

Microsoft Learn: [Introduction](https://learn.microsoft.com/en-us/training/modules/introduction-computer-vision/1-introduction)

### Unit Overview

This unit introduces computer vision as a core area of AI that enables applications to process visual information from images, videos, and live camera feeds.

### Learning Objectives

- Define computer vision.
- Identify common computer vision scenarios.
- Explain why image processing is central to visual AI.
- Recognize that computers process images as digital data.

### Key Terms

| Term | Definition |
| --- | --- |
| Computer vision | AI that processes and analyzes visual information. |
| Visual input | Images, videos, or live camera streams used by AI systems. |
| Digital image | A computer-readable representation of visual information. |

### Lesson Flow

#### 1. What Computer Vision Enables

Explain that computer vision lets software emulate aspects of human visual perception.

#### 2. Common Scenarios

Use examples such as autonomous vehicles, smart checkout systems, and doorbell cameras.

#### 3. Images as Data

Preview that computers process images through numeric values rather than biological sight.

### Suggested Activities

- Ask learners to list three apps that use cameras and identify the likely AI task.
- Have learners match scenarios to computer vision versus non-vision workloads.

### Checks for Understanding

1. What is computer vision?
2. What kinds of input can computer vision process?
3. Why might a smart checkout system use computer vision?
