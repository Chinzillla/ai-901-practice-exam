# Module 5: Introduction to Computer Vision Concepts

## Unit 1 Question Bank: Introduction

---

### 1. What is computer vision?

A. AI that processes visual information  
B. AI that only processes written text  
C. A database backup method  
D. A speech output technique  

**Answer:** A  
**Rationale:** Computer vision focuses on images, videos, and camera feeds.

---

### 2. Which scenario uses computer vision?

A. A doorbell camera detecting a person  
B. A chatbot summarizing a report  
C. A model translating text  
D. A system reading text aloud  

**Answer:** A  
**Rationale:** Person detection from camera input is a visual AI task.

---

### 3. Which input is common for computer vision?

A. Live camera stream  
B. Plain text only  
C. Audio waveform only  
D. SQL query only  

**Answer:** A  
**Rationale:** Computer vision systems commonly process camera feeds.

---

### 4. Why might an autonomous vehicle use computer vision?

A. To detect traffic, pedestrians, and road conditions  
B. To classify email sentiment  
C. To synthesize voice output  
D. To tokenize documents  

**Answer:** A  
**Rationale:** Autonomous vehicles need to interpret visual surroundings.

---

### 5. True or False: Computer vision can work with digital photographs.

**Answer:** True  
**Rationale:** Digital photographs are visual input.

---

### 6. True or False: Computer vision is unrelated to AI.

**Answer:** False  
**Rationale:** Computer vision is a core AI workload.
