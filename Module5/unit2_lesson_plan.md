# Module 5: Introduction to Computer Vision Concepts

## Unit 2: Computer Vision Tasks and Techniques

### Source

Microsoft Learn: [Computer vision tasks and techniques](https://learn.microsoft.com/en-us/training/modules/introduction-computer-vision/2-overview)

### Unit Overview

This unit explains common computer vision tasks: image classification, object detection, semantic segmentation, and contextual image analysis with multimodal models.

### Learning Objectives

- Define image classification.
- Define object detection.
- Explain semantic segmentation.
- Describe contextual image analysis.
- Match vision techniques to practical scenarios.

### Key Terms

| Term | Definition |
| --- | --- |
| Image classification | Predicting a label for the main subject of an image. |
| Object detection | Identifying objects and their locations in an image. |
| Bounding box | A rectangle that marks an object's location. |
| Semantic segmentation | Classifying pixels by the object or region they belong to. |
| Contextual image analysis | Interpreting objects, activities, and relationships in an image. |
| Multimodal model | A model that combines visual and language understanding. |

### Lesson Flow

#### 1. Image Classification

Explain that classification predicts what an image is mainly of.

#### 2. Object Detection

Explain that detection finds multiple objects and their bounding boxes.

#### 3. Semantic Segmentation

Explain that segmentation works at pixel level for more precise object location.

#### 4. Contextual Image Analysis

Explain how multimodal models can describe scenes and suggest tags.

### Suggested Activities

- Sort scenarios into classification, detection, segmentation, or contextual analysis.
- Ask learners to describe what output each technique produces.

### Checks for Understanding

1. What does image classification output?
2. What does object detection add?
3. Why is segmentation more precise?
4. What does a multimodal model help with?
