# Module 8: Select, Deploy, and Evaluate Microsoft Foundry Models

## Unit 2 Question Bank: Explore the model catalog

These questions are authored from the unit's Microsoft Learn concepts and rewritten as exam-style practice items. Answer choices are intentionally paraphrased instead of copied from the lesson text.

---

### 1. A developer wants to find models that can handle text generation, vision, or embeddings. Where should they start?

A. model card  
B. model capability  
C. model selection  
D. The model catalog  

**Answer:** D  
**Rationale:** The requirement matches the unit capability named in the correct option.

### 2. What is the main purpose of "model catalog"?

A. the type of task a model is designed to perform  
B. choosing a model based on task fit, performance, cost, and constraints  
C. a place to browse available models and compare their capabilities  
D. documentation that summarizes a model's purpose, limits, and usage details  

**Answer:** C  
**Rationale:** "model catalog" is a core idea in this unit, and the correct option paraphrases its role.

### 3. A team wants to understand a model's intended use and limitations before deployment. What should they review?

A. Object detection  
B. The model card

**Rationale:** The correct option is the unit concept that best fits the stated requirement.

### 4. Why should learners not pick a model only because it is popular?

A. The best model depends on the workload, constraints, quality needs, and deployment requirements.

### 5. What should a learner remember about "model card"?

A. a configuration task for the interface rather than the AI behavior

### 9. Which three factors matter when selecting a model? Choose three.
