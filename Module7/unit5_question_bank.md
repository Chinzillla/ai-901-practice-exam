# Module 7: Plan and Prepare to Develop AI Solutions on Azure

## Unit 5 Question Bank: Developer tools and SDKs

These questions are authored from the unit's Microsoft Learn concepts and rewritten as exam-style practice items. Answer choices are intentionally paraphrased instead of copied from the lesson text.

---

### 1. A Python app needs to call a deployed AI model from code. What should the developer use?

A. endpoint  
B. key or credential  
C. client application  
D. A supported SDK or API client  

**Answer:** D  
**Rationale:** The requirement matches the unit capability named in the correct option.

---

### 2. What is the main purpose of "SDK"?

A. authentication material used to securely access Azure AI resources  
B. code that sends requests to an AI service and handles responses  
C. a software development kit that helps code interact with a service  
D. the network address an app uses to call a deployed model or service  

**Answer:** C  
**Rationale:** "SDK" is a core idea in this unit, and the correct option paraphrases its role.

---

### 3. An app must know where to send requests for a deployed model. What configuration value is needed?

A. Object detection  
B. The endpoint  
C. SDK  
D. Prompt engineering  

**Answer:** B  
**Rationale:** The correct option is the unit concept that best fits the stated requirement.

### 4. How does using the portal differ from using an SDK?

A. The portal supports interactive setup and testing; an SDK lets application code call the service.  
B. SDKs are only for drawing images.  
C. The portal cannot be used for AI work.  
D. They both remove the need for credentials.  

**Answer:** A  
**Rationale:** The unit distinguishes no-code/portal interaction from code-based integration.

### 5. What should a learner remember about "endpoint"?

A. a configuration task for the interface rather than the AI behavior  
B. a rule-based process that does not use model reasoning  
C. an access-control setting rather than the concept being defined  
D. a callable network address for a deployed model or service  

**Answer:** D  
**Rationale:** "endpoint" is tested by recognizing its purpose, not by memorizing wording from the page.

### 6. Why should credentials not be hard-coded into source files?

A. Hard-coding keys improves security.  
B. AI services never require authentication.  
C. They could be exposed and used to access protected resources.  
D. Credentials only affect font size.  

**Answer:** C  
**Rationale:** Secure credential handling is a core developer concern.

### 7. Which two values are commonly needed by a client app that calls an AI service? Choose two.

D. Callable service address

**Answer:** D, E

### 9. Which three activities are part of developing with AI SDKs? Choose three.

A. Ignore errors  
B. Publish secrets in code
