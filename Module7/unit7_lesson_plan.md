# Module 7: Plan and Prepare to Develop AI Solutions on Azure

## Unit 7: Exercise - Prepare for an AI Development Project

### Source

Microsoft Learn: [Exercise - Prepare for an AI development project](https://learn.microsoft.com/en-us/training/modules/prepare-azure-ai-development/7-exercise-explore-ai-foundry)

### Unit Overview

This exercise unit introduces hands-on exploration of Microsoft Foundry for learners with Azure access. The goal is to connect planning concepts to the Foundry environment.

### Learning Objectives

- Recognize what can be explored in Microsoft Foundry.
- Identify project assets and development resources.
- Reflect on setup needs such as Azure subscription access.
- Connect portal exploration to AI development planning.

### Key Terms

| Term | Definition |
| --- | --- |
| Azure subscription | Azure account context used to create and manage resources. |
| Foundry project | Workspace for AI solution assets. |
| Exercise exploration | Guided hands-on investigation of a platform or tool. |

### Lesson Flow

#### 1. Exercise Orientation

Explain that learners with Azure access can explore Microsoft Foundry directly.

#### 2. Observe Project Concepts

Ask learners to look for projects, resources, models, agents, tools, and endpoints.

#### 3. Debrief

Discuss how the portal supports planning and preparation.

### Suggested Activities

- Learners list the Foundry assets they observe.
- Learners identify one question they would answer before starting a real project.

### Checks for Understanding

1. What access might learners need for the exercise?
2. Which Foundry assets should learners look for?
3. How does exploration support planning?
