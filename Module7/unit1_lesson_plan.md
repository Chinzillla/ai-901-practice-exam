# Module 7: Plan and Prepare to Develop AI Solutions on Azure

## Unit 1: Introduction

### Source

Microsoft Learn: [Introduction](https://learn.microsoft.com/en-us/training/modules/prepare-azure-ai-development/1-introduction)

### Unit Overview

This unit introduces the planning mindset needed for AI application development on Azure. Learners review why modern AI solutions often combine models, services, prompt engineering, and custom code, and why Microsoft Foundry is introduced as a platform for AI development.

### Learning Objectives

- Explain why developers need to plan AI solutions carefully.
- Recognize that AI solutions can combine models, AI services, prompts, and custom code.
- Identify Microsoft Foundry as an Azure AI development platform.
- Explain why service, tool, and framework choices matter before development starts.

### Key Terms

| Term | Definition |
| --- | --- |
| AI solution | An application or system that uses AI capabilities to solve a problem. |
| Prompt engineering | Designing prompts to guide generative AI model behavior. |
| Microsoft Foundry | A platform for AI development on Microsoft Azure. |
| AI service | A prebuilt capability or hosted resource that supports AI workloads. |

### Lesson Flow

#### 1. Why Planning Matters

Explain that AI projects can include multiple components, including machine learning models, AI services, prompt engineering, and application code.

#### 2. Azure AI Development Options

Explain that Azure offers multiple services and tools, so developers should evaluate which options fit the project.

#### 3. Microsoft Foundry Preview

Introduce Microsoft Foundry as a comprehensive platform for developing AI solutions on Azure.

### Suggested Activities

- Learners list components that might be needed in a generative AI application.
- Learners identify why a project should choose tools before implementation.

### Checks for Understanding

1. What components can an AI solution include?
2. Why should developers plan service choices early?
3. What platform is introduced for AI development on Azure?
