# Module 7: Plan and Prepare to Develop AI Solutions on Azure

## Unit 8: Module Assessment Prep

### Source

Microsoft Learn: [Module assessment](https://learn.microsoft.com/en-us/training/modules/prepare-azure-ai-development/8-knowledge-check)

### Unit Overview

This unit prepares learners for the Module 7 assessment without copying Microsoft Learn assessment items. It reviews Microsoft Foundry, Foundry Tools, developer tooling, AI capabilities, and responsible AI.

### Learning Objectives

- Review Microsoft Foundry concepts.
- Identify Foundry Tools and their use cases.
- Recognize developer tools and SDKs.
- Apply responsible AI principles to development scenarios.

### Key Terms

| Term | Definition |
| --- | --- |
| Assessment prep | Original review before a module assessment. |
| Tool selection | Choosing the right service, portal, extension, API, or SDK for a task. |

### Lesson Flow

#### 1. Review Foundry

Review projects, resources, portal, SDK, models, agents, tools, and knowledge.

#### 2. Review Foundry Tools

Review Language, Speech, Translator, Document Intelligence, and Content Understanding.

#### 3. Review Developer Tooling

Review VS Code, Foundry Toolkit, GitHub, Copilot, SDKs, and APIs.

### Suggested Activities

- Learners complete a tool-selection table.
- Learners answer scenario questions and explain their choices.

### Checks for Understanding

1. Which portal manages Foundry projects?
2. Which component provides prebuilt AI capabilities?
3. Which VS Code extension supports Foundry workflows?
