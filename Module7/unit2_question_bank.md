# Module 7: Plan and Prepare to Develop AI Solutions on Azure

## Unit 2 Question Bank: What is AI?

These questions are authored from the unit's Microsoft Learn concepts and rewritten as exam-style practice items. Answer choices are intentionally paraphrased instead of copied from the lesson text.

---

### 1. A system predicts whether a customer will renew based on historical records. Which AI approach is involved?

A. artificial intelligence  
B. generative AI  
C. AI workload  
D. Machine learning prediction  

**Answer:** D  
**Rationale:** The requirement matches the unit capability named in the correct option.

---

### 2. What is the main purpose of "artificial intelligence"?

A. AI that creates new content in response to prompts  
B. a category of AI solution such as language, speech, vision, or extraction  
C. software capability that imitates human-like perception, reasoning, or decision support  
D. training models from data so they can make predictions or classifications  

**Answer:** C  
**Rationale:** "artificial intelligence" is a core idea in this unit, and the correct option paraphrases its role.

### 3. A user asks a model to draft a support email from bullet points. Which AI category fits?

A. Object detection  
B. Content generation from prompts  
C. machine learning  
D. Prompt engineering  

**Answer:** B  
**Rationale:** The correct option is the unit concept that best fits the stated requirement.

### 4. How is generative AI different from predictive machine learning?

A. Generative AI creates new content; predictive ML estimates labels or values from data.  
B. Predictive ML only creates images.  
C. Generative AI cannot use prompts.  
D. They are unrelated to Azure AI solutions.  

**Answer:** A  
**Rationale:** The unit introduces common AI categories and how they differ.

### 5. What should a learner remember about "machine learning"?

A. a configuration task for the interface rather than the AI behavior  
B. a rule-based process that does not use model reasoning  
C. an access-control setting rather than the concept being defined  
D. learning from data so they can make predictions or classifications  

**Answer:** D  
**Rationale:** "machine learning" is tested by recognizing its purpose, not by memorizing wording from the page.

### 6. Why should a developer identify the AI workload before building?

A. Workload selection only changes the billing currency.  
B. The workload has no effect on design choices.

**Rationale:** AI-901 scenarios often test matching requirements to the correct workload.

### 9. Which three capabilities can AI solutions provide? Choose three.

### 10. Which description best matches "generative AI"?
