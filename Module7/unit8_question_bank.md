# Module 7: Plan and Prepare to Develop AI Solutions on Azure

## Unit 8 Question Bank: Module Assessment Prep

---

### 1. Which portal is used to work visually with Microsoft Foundry projects?

A. Microsoft Foundry portal  
B. Local File Explorer only  
C. Printer portal  
D. Disk Manager  

**Answer:** A  
**Rationale:** The Foundry portal is the web interface for projects and assets.

---

### 2. Which component provides prebuilt services for common AI tasks?

A. Foundry Tools  
B. Foundry IQ only  
C. Git history  
D. Visual Studio theme settings  

**Answer:** A  
**Rationale:** Foundry Tools provide prebuilt APIs and models.

---

### 3. Which VS Code extension supports Foundry project workflows?

A. Foundry Toolkit  
B. Python extension only  
C. Markdown preview only  
D. Spell checker only  

**Answer:** A  
**Rationale:** The Foundry Toolkit supports browsing and managing Foundry assets.

---

### 4. Which tool supports text analysis?

A. Azure Language  
B. Azure Speech only  
C. Azure Translator only  
D. GitHub only  

**Answer:** A  
**Rationale:** Azure Language supports NLP and text analysis tasks.

---

### 5. Which principle addresses making users aware of AI use and limitations?

A. Transparency  
B. Object detection  
C. OCR  
D. Audio capture  

**Answer:** A  
**Rationale:** Transparency helps users understand AI systems.

---

### 6. True or False: The Foundry SDK supports programmatic automation.

**Answer:** True  
**Rationale:** The SDK can automate project operations through code.
