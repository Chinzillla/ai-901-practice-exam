# Module 7: Plan and Prepare to Develop AI Solutions on Azure

## Unit 1 Question Bank: Introduction

---

### 1. Why should developers plan AI solutions before building?

A. AI solutions can combine models, services, prompts, and custom code  
B. Planning removes the need for testing  
C. AI applications never use services  
D. Azure has only one AI option  

**Answer:** A  
**Rationale:** AI solutions often require multiple coordinated components.

---

### 2. Which platform is introduced for AI development on Azure?

A. Microsoft Foundry  
B. Microsoft Paint  
C. Windows Explorer  
D. Task Manager  

**Answer:** A  
**Rationale:** Microsoft Foundry is presented as a platform for AI development on Azure.

---

### 3. Which item can be part of an AI development project?

A. Prompt engineering  
B. Printer ink only  
C. Cable labeling only  
D. Manual filing only  

**Answer:** A  
**Rationale:** Prompt engineering is important in generative AI solutions.

---

### 4. True or False: AI solutions may use both AI services and custom code.

**Answer:** True  
**Rationale:** Many AI applications combine prebuilt services with application logic.
