# Module 7: Plan and Prepare to Develop AI Solutions on Azure

## Unit 5: Developer Tools and SDKs

### Source

Microsoft Learn: [Developer tools and SDKs](https://learn.microsoft.com/en-us/training/modules/prepare-azure-ai-development/5-tools-and-sdks)

### Unit Overview

This unit introduces development environments, the Foundry Toolkit extension for Visual Studio Code, GitHub, GitHub Copilot, supported programming languages, APIs, and SDKs for building AI applications on Azure.

### Learning Objectives

- Identify common development environments for Azure AI.
- Explain the role of the Foundry Toolkit extension.
- Describe how GitHub and GitHub Copilot support development.
- Identify common languages, APIs, and SDKs for AI applications.

### Key Terms

| Term | Definition |
| --- | --- |
| IDE | Integrated development environment, such as Visual Studio. |
| VS Code | Visual Studio Code, a code editor used for many development workflows. |
| Foundry Toolkit | VS Code extension for working with Foundry projects and assets. |
| GitHub | Source control and DevOps platform. |
| GitHub Copilot | AI coding assistant. |
| SDK | Software development kit for working with services in code. |
| REST API | Web API style commonly used to call cloud services. |

### Lesson Flow

#### 1. Development Environments

Compare Visual Studio and VS Code as examples of development tools.

#### 2. Foundry Toolkit

Review browsing resources, deploying models, testing models and agents, configuring agents, and generating integration code.

#### 3. GitHub and Copilot

Explain source control, DevOps, and AI-assisted development support.

#### 4. Languages and SDKs

Review C#, Python, Node, TypeScript, Java, Foundry SDK, OpenAI API, Foundry Tools SDKs, and REST APIs.

### Suggested Activities

- Learners match development tools to tasks.
- Learners choose a toolchain for a sample AI app.

### Checks for Understanding

1. What does the Foundry Toolkit help with?
2. Why is GitHub useful in team development?
3. What is an SDK?
4. Which languages can be used for AI apps?
