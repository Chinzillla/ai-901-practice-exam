# Module 6: Introduction to AI-Powered Information Extraction Concepts

## Unit 5 Question Bank: Exercise - Explore AI Information Extraction

---

### 1. In receipt extraction, what does OCR provide?

A. Text detected from the receipt image  
B. Spoken audio output  
C. Object labels only  
D. Prompt temperature settings  

**Answer:** A  
**Rationale:** OCR reads visual text from the receipt.

---

### 2. What can an LLM help do after OCR?

A. Interpret text and map it to fields  
B. Remove all extracted text  
C. Convert the receipt into music  
D. Ignore the schema  

**Answer:** A  
**Rationale:** Generative AI can help semantically match values to fields.

---

### 3. Which field is common in receipt extraction?

A. Total amount  
B. Audio pitch  
C. Camera frame rate  
D. Token vocabulary size only  

**Answer:** A  
**Rationale:** Receipt processing commonly extracts totals.

---

### 4. True or False: Receipt extraction can combine OCR and LLM reasoning.

**Answer:** True  
**Rationale:** OCR reads text, while an LLM can interpret fields.
