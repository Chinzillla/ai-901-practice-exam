# Module 6: Introduction to AI-Powered Information Extraction Concepts

## Unit 3: Optical Character Recognition (OCR)

### Source

Microsoft Learn: [Optical character recognition (OCR)](https://learn.microsoft.com/en-us/training/modules/introduction-information-extraction/3-vision-extraction)

### Unit Overview

This unit explains OCR as the process of converting visual text from images, scanned documents, screenshots, forms, and notes into editable and searchable text.

### Learning Objectives

- Define OCR.
- Identify OCR source types.
- Describe the OCR pipeline stages.
- Explain why image quality affects OCR accuracy.

### Key Terms

| Term | Definition |
| --- | --- |
| OCR | Optical character recognition. |
| Image acquisition | Providing the image input to the OCR system. |
| Preprocessing | Enhancing the image for better recognition. |
| Text region detection | Locating areas that contain text. |
| Character recognition | Identifying characters in detected text regions. |
| Post-processing | Producing usable text with formatting or corrections. |

### Lesson Flow

#### 1. OCR Sources

Review scanned invoices, receipts, PDFs with images, screenshots, forms, and handwritten notes.

#### 2. OCR Pipeline

Teach the five stages: acquisition, preprocessing, region detection, recognition, and output/post-processing.

#### 3. Preprocessing

Explain noise reduction, contrast adjustment, skew correction, and resolution optimization.

#### 4. Detection and Recognition

Explain layout analysis, text blocks, reading order, character classification, context analysis, and confidence scoring.

### Suggested Activities

- Learners order OCR pipeline stages.
- Learners identify OCR challenges in sample document photos.

### Checks for Understanding

1. What does OCR do?
2. Why is image quality important?
3. What is text region detection?
4. What is post-processing?
