# Module 6: Introduction to AI-Powered Information Extraction Concepts

## Unit 1: Introduction

### Source

Microsoft Learn: [Introduction](https://learn.microsoft.com/en-us/training/modules/introduction-information-extraction/1-introduction)

### Unit Overview

This unit introduces AI-powered information extraction as the process of extracting structured fields from unstructured media such as documents, images, video, and audio.

### Learning Objectives

- Define AI information extraction.
- Identify common extraction scenarios in finance, legal, healthcare, and logistics.
- Explain why extracted fields support workflow automation.

### Key Terms

| Term | Definition |
| --- | --- |
| Information extraction | Identifying and extracting relevant fields and values from unstructured content. |
| Structured data | Data organized into defined fields. |
| Unstructured media | Content such as documents, images, audio, or video that is not already fielded. |

### Lesson Flow

#### 1. What Information Extraction Does

Explain that extraction turns unstructured content into useful structured data.

#### 2. Common Document Scenarios

Review invoices, receipts, financial statements, contracts, tax forms, medical records, shipping documents, and purchase orders.

#### 3. Automation Value

Explain that extracted fields can feed business workflows and reduce manual processing.

### Suggested Activities

- Learners list fields that could be extracted from an invoice.
- Learners match industries to likely extraction documents.

### Checks for Understanding

1. What is information extraction?
2. What is an example of unstructured media?
3. Why is extraction useful in business workflows?
