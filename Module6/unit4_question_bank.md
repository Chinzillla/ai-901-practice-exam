# Module 6: Introduction to AI-Powered Information Extraction Concepts

## Unit 4 Question Bank: Field extraction and mapping

These questions are authored from the unit's Microsoft Learn concepts and rewritten as exam-style practice items. Answer choices are intentionally paraphrased instead of copied from the lesson text.

---

### 1. An invoice solution must return vendor name, invoice date, and total as separate values. Which capability is needed?

A. field mapping  
B. custom extraction  
C. confidence score  
D. Field extraction  

**Answer:** D  
**Rationale:** The requirement matches the unit capability named in the correct option.

---

### 2. What is the main purpose of "field extraction"?

A. training or configuring extraction for fields that are specific to a business form  
B. an indicator of how certain the system is about an extracted value  
C. identifying specific named values in content  
D. connecting extracted values to expected output fields  

**Answer:** C  
**Rationale:** "field extraction" is a core idea in this unit, and the correct option paraphrases its role.

---

### 3. A system assigns the text '05/06/2026' to the InvoiceDate property in an output record. What is this?

A. Speech synthesis  
B. Mapping values to output fields  
C. Prompt engineering  
D. Object detection  

**Answer:** B  
**Rationale:** The correct option is the unit concept that best fits the stated requirement.

---

### 4. How is field extraction different from OCR?

A. OCR reads text; field extraction identifies which text is the specific value a workflow needs.  
B. Field extraction generates synthetic voices.  
C. OCR maps every value to a schema automatically.  
D. They both only classify image labels.  

**Answer:** A  
**Rationale:** The unit separates reading text from mapping meaningful fields.

---

### 5. What should a learner remember about "field mapping"?

A. a configuration task for the interface rather than the AI behavior  
B. a rule-based process that does not use model reasoning  
C. an access-control setting rather than the concept being defined  
D. mapping extracted values to expected output fields  

**Answer:** D  
**Rationale:** "field mapping" is tested by recognizing its purpose, not by memorizing wording from the page.

---

### 6. Why is mapping important after extracting fields?

A. It converts documents into spoken prompts.  
B. It hides all extracted data from users.  
C. It lets applications store each value in the correct business field.  
D. It removes the need to know the document type.  

**Answer:** C  
**Rationale:** Extraction becomes useful when values are mapped to structured outputs.

### 7. Which two examples are document fields? Choose two.

A. Camera shutter speed  
B. Voice pitch  
C. Prompt creativity setting  
D. Purchase order number  
E. Due date  

**Answer:** D, E  
**Rationale:** Business documents often contain identifiers and dates that can be extracted.

### 8. A human reviewer checks low-confidence extracted totals before submission. What value triggered review?

A. Extraction certainty score  
B. Information extraction  
C. Model evaluation  
D. Choose a model based on popularity instead of task fit.  

**Answer:** A  
**Rationale:** The scenario points to the unit capability described by the correct option.

### 9. Which three practices support reliable field extraction? Choose three.

A. Ignore form layout  
B. Treat every document as an image-generation prompt  
C. Remove all labels before testing  
D. Define expected fields  
E. Review low-confidence values  
F. Validate mapped output  

**Answer:** D, E, F  
**Rationale:** Good extraction depends on clear fields, confidence-aware review, and validation.

### 10. Which description best matches "custom extraction"?

A. Use a visual analysis capability when the input is only text.  
B. Handle sensitive data without the required access controls.  
C. adapting or configuring extraction for fields that are specific to a business form  
D. Rely on one successful trial instead of broader evaluation.  

**Answer:** C  
**Rationale:** The correct answer paraphrases how the unit explains "custom extraction".
