# Module 6: Introduction to AI-Powered Information Extraction Concepts

## Unit 7 Question Bank: Summary

---

### 1. What is the goal of information extraction?

A. Store useful extracted data in structured formats  
B. Remove all text from documents  
C. Generate unrelated images  
D. Convert speech into music  

**Answer:** A  
**Rationale:** Extraction makes unstructured content usable in structured systems.

---

### 2. Which technique converts visual text into text data?

A. OCR  
B. Speech synthesis  
C. Image generation  
D. Object detection only  

**Answer:** A  
**Rationale:** OCR reads text from images.

---

### 3. Which process maps extracted text to labeled fields?

A. Field extraction  
B. Speech recognition  
C. Image classification  
D. Prompt tokenization only  

**Answer:** A  
**Rationale:** Field extraction assigns business meaning to values.

---

### 4. True or False: Extracted values may need normalization before use.

**Answer:** True  
**Rationale:** Dates, currencies, and numbers often need standardized formats.
