# Module 6: Introduction to AI-Powered Information Extraction Concepts

## Unit 6: Module Assessment Prep

### Source

Microsoft Learn: [Module assessment](https://learn.microsoft.com/en-us/training/modules/introduction-information-extraction/7-knowledge-check)

### Unit Overview

This unit prepares learners for the Module 6 assessment without copying Microsoft Learn assessment items. It reviews information extraction, OCR, generative AI-assisted field mapping, validation, and integration.

### Learning Objectives

- Review AI information extraction.
- Explain OCR's role.
- Explain how generative AI can enhance extraction.
- Match extraction techniques to scenarios.

### Key Terms

| Term | Definition |
| --- | --- |
| Assessment prep | Original review before a knowledge check. |
| Semantic mapping | Matching extracted text to business meaning. |

### Lesson Flow

#### 1. Review Extraction

Review unstructured content to structured fields.

#### 2. Review OCR

Review OCR as visual text recognition.

#### 3. Review Field Mapping

Review templates, machine learning, LLMs, normalization, and validation.

### Suggested Activities

- Learners explain OCR versus field extraction.
- Learners choose extraction approaches for invoices, receipts, and contracts.

### Checks for Understanding

1. What does information extraction produce?
2. What does OCR convert?
3. How can generative AI help extraction?
