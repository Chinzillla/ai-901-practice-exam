# Module 6: Introduction to AI-Powered Information Extraction Concepts

## Unit 3 Question Bank: Optical character recognition (OCR)

These questions are authored from the unit's Microsoft Learn concepts and rewritten as exam-style practice items. Answer choices are intentionally paraphrased instead of copied from the lesson text.

---

### 1. A system needs to read text from photos of street signs. Which capability is required?

A. printed text recognition  
B. handwriting recognition  
C. text location  
D. OCR  

**Answer:** D  
**Rationale:** This scenario matches Optical character recognition (OCR) as described in the unit.

---

### 2. What is the main purpose of "OCR"?

A. recognizing handwritten characters or words  
B. identifying where recognized text appears in an image  
C. detecting and reading text from images or scanned documents  
D. reading machine-printed characters from visual content  

**Answer:** C  
**Rationale:** "OCR" is a core idea in this unit, and the correct option paraphrases its role.

---

### 3. A document processor extracts words and their positions from a scanned page. What OCR output is useful here?

A. Speech synthesis  
B. Recognized text with locations  
C. Prompt engineering  
D. Object detection  

**Answer:** B  
**Rationale:** The correct option is the unit concept that best fits the stated requirement.

---

### 4. How is OCR different from general object detection?

A. OCR reads text characters; object detection identifies and locates visual objects.  
B. OCR creates voices, while object detection summarizes text.  
C. Object detection only works on PDFs.  
D. OCR is the same as sentiment analysis.  

**Answer:** A  
**Rationale:** OCR is specialized for visual text extraction.

---

### 5. What should a learner remember about "printed text recognition"?

A. a process for changing the visual theme of the development portal  
B. a manual-only activity that does not use an AI model  
C. a security shortcut that removes the need for access controls  
D. reading machine-printed characters from visual content for this scenario  

**Answer:** D  
**Rationale:** "printed text recognition" is tested by recognizing its purpose, not by memorizing wording from the page.

---

### 6. Why can image quality affect OCR results?

A. OCR requires audio quality instead.  
B. Low contrast always improves OCR accuracy.  
C. Blur, skew, low contrast, or poor resolution can make characters harder to recognize.  
D. Image quality never affects text recognition.  

**Answer:** C  
**Rationale:** OCR depends on readable visual text.

---

### 7. Which two inputs are suitable for OCR? Choose two.

A. A spoken phone call only  
B. A sentiment label  
C. A calendar invitation with no image  
D. A scanned invoice  
E. A photo of a sign  

**Answer:** D, E  
**Rationale:** OCR works on visual content that contains text.

---

### 8. A claims app reads handwritten notes on a form. Which OCR challenge applies?

A. Handwriting recognition for this scenario  
B. Information extraction  
C. Model evaluation  
D. Choose a model only because it has the largest name recognition.  

**Answer:** A  
**Rationale:** The scenario points to the unit capability described by the correct option.

---

### 9. Which three factors can affect OCR quality? Choose three.

A. Voice cadence  
B. Prompt temperature only  
C. Chat history length  
D. Image resolution  
E. Text clarity  
F. Page orientation  

**Answer:** D, E, F  
**Rationale:** Readable, well-oriented images support better OCR.
