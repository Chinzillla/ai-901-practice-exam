# Module 6: Introduction to AI-Powered Information Extraction Concepts

## Unit 6 Question Bank: Module Assessment Prep

---

### 1. Which statement best describes information extraction?

A. Analyzing unstructured content to identify fields and values  
B. Querying a structured database with SQL only  
C. Copying cloud files to a local disk  
D. Reading text aloud  

**Answer:** A  
**Rationale:** Extraction finds useful structured data in unstructured sources.

---

### 2. What does OCR convert?

A. Images of text into machine-readable text  
B. Text into spoken audio  
C. Pixels into generated art only  
D. Tokens into embeddings only  

**Answer:** A  
**Rationale:** OCR recognizes visual text.

---

### 3. How can generative AI enhance extraction?

A. By using semantic understanding to match values to fields  
B. By manually coding every template  
C. By removing all document context  
D. By generating unrelated documents only  

**Answer:** A  
**Rationale:** LLMs can interpret text against a schema.

---

### 4. True or False: Field extraction is the same as OCR.

**Answer:** False  
**Rationale:** OCR reads text; field extraction maps text to business meaning.
