# Module 6: Introduction to AI-Powered Information Extraction Concepts

## Unit 2 Question Bank: Overview of information extraction

These questions are authored from the unit's Microsoft Learn concepts and rewritten as exam-style practice items. Answer choices are intentionally paraphrased instead of copied from the lesson text.

---

### 1. A company receives many scanned claims and wants key values entered into a system automatically. Which workload is this?

A. structured data  
B. unstructured content  
C. content understanding  
D. Information extraction  

**Answer:** D  
**Rationale:** The requirement matches the unit capability named in the correct option.

---

### 2. What is the main purpose of "information extraction"?

A. content such as documents, images, audio, or video without fixed fields  
B. AI analysis that identifies useful information in varied content types  
C. turning content into structured data that can be searched, stored, or automated  
D. organized values such as fields, labels, or records  

**Answer:** C  
**Rationale:** "information extraction" is a core idea in this unit, and the correct option paraphrases its role.

### 3. A contract is a long PDF with clauses, dates, and parties. What kind of source content is it before extraction?

A. Speech synthesis  
B. Unstructured or semi-structured content  
C. Prompt engineering  
D. Object detection  

**Answer:** B  
**Rationale:** The correct option is the unit concept that best fits the stated requirement.

### 4. How is information extraction different from simple document storage?

A. Extraction identifies usable values; storage only keeps the file.  
B. Extraction only changes file names.  
C. Storage always understands every field automatically.  
D. Extraction is used only for speech synthesis.  

**Answer:** A  
**Rationale:** The unit focuses on turning content into actionable data.

### 5. What should a learner remember about "structured data"?

A. a configuration task for the interface rather than the AI behavior  
B. a rule-based process that does not use model reasoning  
C. an access-control setting rather than the concept being defined  
D. structured values such as fields, labels, or records  

**Answer:** D  
**Rationale:** "structured data" is tested by recognizing its purpose, not by memorizing wording from the page.

### 6. Why is a schema or expected field list useful in extraction?

A. It is only used to generate images.  
B. It removes the need to validate outputs.  
C. It defines what information the solution should find and return.  
D. It prevents the system from reading text.  

**Answer:** C  
**Rationale:** Extraction depends on knowing which fields or data points matter.

### 7. Which two are common information extraction targets? Choose two.

A. Display settings  
B. Client input setting  
C. Audio volume preference only  
D. Document fields  
E. Text in images  

**Answer:** D, E  
**Rationale:** Document values and image text are common extraction targets.

### 8. A pipeline creates records from documents so downstream apps can filter and report on them. What is the desired output?

A. Organized fields and records  
B. Model evaluation  
C. Choose a model based on popularity instead of task fit.  
D. Rely on one successful trial instead of broader evaluation.  

**Answer:** A  
**Rationale:** The scenario points to the unit capability described by the correct option.

---

### 9. Which three benefits can information extraction provide? Choose three.

A. Guaranteed perfect results without review  
B. Automatic model fairness  
C. Visual style generation  
D. Reduced manual data entry  
E. Searchable structured records  
F. Automation from document content  

**Answer:** D, E, F  
**Rationale:** Extraction makes content usable in workflows, but validation still matters.

### 10. Which description best matches "unstructured content"?

C. unstructured material such as documents, images, audio, or video without fixed fields  
D. Use a visual analysis capability when the input is only text.  

**Answer:** C  
**Rationale:** The correct answer paraphrases how the unit explains "unstructured content".
