# Module 1: Introduction to AI Concepts

## Unit 5 Question Bank: Computer vision

These questions are authored from the unit's Microsoft Learn concepts and rewritten as exam-style practice items. Answer choices are intentionally paraphrased instead of copied from the lesson text.

---

### 1. An app must identify whether a photo contains a receipt, a license, or a passport. Which task is this?

A. computer vision  
B. object detection  
C. image generation  
D. Image classification  

**Answer:** D  
**Rationale:** The requirement matches the unit capability named in the correct option.

---

### 2. What is the main purpose of "computer vision"?

A. finding objects in an image and locating them with regions or boxes  
B. creating new images from prompts or other inputs  
C. AI that interprets information in images and video  
D. assigning a label to an image based on its main content  

**Answer:** C  
**Rationale:** "computer vision" is a core idea in this unit, and the correct option paraphrases its role.

---

### 3. A warehouse camera must count each visible package and mark where each one appears. Which capability is needed?

A. Information extraction  
B. Visual object localization  
C. Prompt engineering  
D. Speech synthesis  

**Answer:** B  
**Rationale:** The correct option is the unit concept that best fits the stated requirement.

---

### 4. How does object detection differ from image classification?

A. Detection identifies and locates objects; classification assigns an overall label.  
B. Detection summarizes text, while classification translates speech.  
C. Classification always creates new images.  
D. Both techniques only redact personal information.  

**Answer:** A  
**Rationale:** The unit separates whole-image labeling from locating individual objects.

---

### 5. What should a learner remember about "image classification"?

A. a configuration task for the interface rather than the AI behavior  
B. a rule-based process that does not use model reasoning  
C. an access-control setting rather than the concept being defined  
D. labeling by a label to an image based on its main content  

**Answer:** D  
**Rationale:** "image classification" is tested by recognizing its purpose, not by memorizing wording from the page.

---

### 6. Why do computer vision systems process images as numerical data?

A. Numerical data prevents model testing.  
B. It turns every image into speech output.  
C. Models need pixel values and visual features they can analyze mathematically.  
D. Only text data can be used by AI models.  

**Answer:** C  
**Rationale:** Vision models work with numeric image representations and learned visual features.

### 7. Which two are computer vision tasks? Choose two.

A. Extracting sentiment from a review  
B. Synthesizing a spoken response  
C. Choosing an email subject line  
D. Detecting objects in images  
E. Classifying an image by content  

**Answer:** D, E  
**Rationale:** Both correct answers require interpreting visual input.

### 9. Which three capabilities belong to computer vision scenarios? Choose three.

A. Speech recognition

### 10. Which description best matches "object detection"?
