# Module 1: Introduction to AI Concepts

## Unit 9: Module Assessment Prep

### Source

Microsoft Learn: [Module assessment](https://learn.microsoft.com/en-us/training/modules/get-started-ai-fundamentals/8-knowledge-check)

### Unit Overview

This unit prepares learners for the module assessment without copying Microsoft Learn assessment items. Learners review the major concepts from Module 1 and practice identifying the AI workload or principle that best fits a scenario.

### Learning Objectives

By the end of this unit, learners should be able to:

- Review the major AI workloads introduced in Module 1.
- Match generative AI, NLP, speech, vision, information extraction, and responsible AI to scenarios.
- Distinguish between similar capabilities such as speech recognition and speech synthesis.
- Explain why responsible AI principles apply across AI solutions.

### Key Terms

| Term | Definition |
| --- | --- |
| Assessment prep | Review and practice designed to prepare learners for a knowledge check. |
| Scenario recognition | Identifying the correct concept or capability from a described requirement. |
| Workload selection | Choosing the AI workload that best fits a user or business need. |

### Lesson Flow

#### 1. Review the Module

Review each workload:

- Generative AI and agents
- Text and natural language
- Speech
- Computer vision
- Information extraction
- Responsible AI

#### 2. Practice Scenario Recognition

Give learners short requirements and ask them to choose the best workload. Require a brief explanation for each answer.

#### 3. Review Common Confusions

Clarify:

- Speech recognition versus speech synthesis.
- Image classification versus object detection.
- OCR versus field extraction.
- Content filters versus full responsible AI practice.

### Suggested Activities

#### Activity 1: One-Minute Workload Review

Learners explain each workload in one sentence.

#### Activity 2: Scenario Round

Learners answer rapid scenario questions and justify their choices.

#### Activity 3: Confidence Check

Learners mark concepts as strong, needs review, or unclear.

### Checks for Understanding

1. Which workload creates new content?
2. Which workload analyzes text?
3. Which workload converts speech to text?
4. Which workload identifies objects in images?
5. Which capability extracts fields from documents?
6. Why does responsible AI matter?
