# Module 1: Introduction to AI Concepts

## Unit 9 Question Bank: Module Assessment Prep

These original questions prepare learners for Module 1 assessment concepts without copying Microsoft Learn assessment items.

---

### 1. A user enters a prompt and an AI system writes a new paragraph. Which workload is involved?

A. Generative AI  
B. Computer vision  
C. Speech recognition  
D. OCR only  

**Answer:** A  
**Rationale:** Generative AI creates new content in response to prompts.

---

### 2. A system analyzes reviews to determine whether they are positive or negative. Which workload is involved?

A. Natural language processing  
B. Speech synthesis  
C. Object detection  
D. Facial recognition  

**Answer:** A  
**Rationale:** Sentiment analysis is a text analysis capability within NLP.

---

### 3. A system converts a recorded call into text. Which capability is involved?

A. Speech recognition  
B. Speech synthesis  
C. Image classification  
D. Semantic segmentation  

**Answer:** A  
**Rationale:** Speech recognition converts spoken input into text.

---

### 4. A system reads a generated answer aloud. Which capability is involved?

A. Speech synthesis  
B. Speech recognition  
C. Object detection  
D. Key-term extraction  

**Answer:** A  
**Rationale:** Speech synthesis converts text into spoken audio.

---

### 5. A system labels an uploaded image as a "laptop." Which computer vision task is involved?

A. Image classification  
B. Speech translation  
C. PII redaction  
D. Text summarization  

**Answer:** A  
**Rationale:** Image classification predicts an overall image label.

---

### 6. A system locates each product in a shelf photo. Which computer vision task is involved?

A. Object detection  
B. Language detection  
C. Speech synthesis  
D. Sentiment analysis  

**Answer:** A  
**Rationale:** Object detection identifies where objects are located.

---

### 7. A system extracts invoice numbers and totals from scanned invoices. Which workload is involved?

A. Information extraction  
B. Image generation  
C. Speech translation  
D. Responsible AI only  

**Answer:** A  
**Rationale:** Extracting specific fields from documents is information extraction.

---

### 8. A bank discloses that an AI system is used in loan decisions. Which responsible AI principle is most relevant?

A. Transparency  
B. Image classification  
C. Speech recognition  
D. Tokenization  

**Answer:** A  
**Rationale:** Transparency involves clear disclosure about AI use.

---

### 9. Which principle helps ensure an AI system works for users with different abilities?

A. Inclusiveness  
B. File compression  
C. Visual search  
D. OCR  

**Answer:** A  
**Rationale:** Inclusiveness focuses on designing systems that serve diverse users.

---

### 10. True or False: Responsible AI applies only after an AI system is deployed.

**Answer:** False  
**Rationale:** Responsible AI should be considered from planning through operation.

---

### 11. True or False: A single application can use both speech recognition and generative AI.

**Answer:** True  
**Rationale:** A voice assistant may convert speech to text and then generate an answer.

---

### 12. True or False: OCR can help extract text from an image of a document.

**Answer:** True  
**Rationale:** OCR identifies text in image-based sources.
