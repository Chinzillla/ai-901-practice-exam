# Module 1: Introduction to AI Concepts

## Unit 10 Question Bank: Summary

These questions provide cumulative review for Module 1.

---

### 1. Which workload creates new text, images, code, or other content?

A. Generative AI  
B. OCR only  
C. File compression  
D. Network routing  

**Answer:** A  
**Rationale:** Generative AI creates new content from prompts or instructions.

---

### 2. Which workload analyzes written language for meaning, entities, or sentiment?

A. Natural language processing  
B. Computer vision  
C. Speech synthesis only  
D. Disk formatting  

**Answer:** A  
**Rationale:** NLP focuses on understanding and analyzing language.

---

### 3. Which workload converts spoken input into text?

A. Speech recognition  
B. Image classification  
C. Responsible AI  
D. Field extraction only  

**Answer:** A  
**Rationale:** Speech recognition commonly appears as speech-to-text.

---

### 4. Which workload analyzes images, videos, or camera feeds?

A. Computer vision  
B. Sentiment analysis  
C. Text-to-speech  
D. Key-term extraction  

**Answer:** A  
**Rationale:** Computer vision analyzes visual input.

---

### 5. Which workload can extract serial numbers or invoice totals from documents and images?

A. Information extraction  
B. Speech synthesis  
C. Image generation only  
D. Manual sorting  

**Answer:** A  
**Rationale:** Information extraction finds useful values in unstructured sources.

---

### 6. Which concept focuses on building trustworthy AI systems with safeguards?

A. Responsible AI  
B. File indexing only  
C. Speech translation  
D. Image resizing  

**Answer:** A  
**Rationale:** Responsible AI includes principles and guardrails that reduce risk.

---

### 7. A voice assistant listens to a user and speaks back. Which workloads may be combined?

A. Speech recognition, generative AI, and speech synthesis  
B. Disk cleanup and file compression  
C. Object detection only  
D. OCR only  

**Answer:** A  
**Rationale:** The assistant must understand spoken input, generate or retrieve a response, and speak it aloud.

---

### 8. A system identifies where each person appears in a photo. Which task fits best?

A. Object detection  
B. Speech recognition  
C. Language detection  
D. Text summarization  

**Answer:** A  
**Rationale:** Object detection locates objects or people in images.

---

### 9. True or False: Responsible AI applies across multiple AI workloads.

**Answer:** True  
**Rationale:** Responsible AI applies to generative AI, agents, vision, speech, extraction, and more.

---

### 10. True or False: Module 1 introduces only one AI capability.

**Answer:** False  
**Rationale:** Module 1 introduces several AI workloads and responsible AI concepts.

---

### 11. True or False: AI agents can use tools and respond to context.

**Answer:** True  
**Rationale:** Agents combine models, instructions, and tools to assist users and perform tasks.

---

### 12. True or False: Computer vision and NLP solve the same type of input problem.

**Answer:** False  
**Rationale:** Computer vision works with visual input, while NLP works with language.
