# Module 1: Introduction to AI Concepts

## Unit 8 Question Bank: Exercise - Explore AI Workloads

These questions reinforce the exercise goal of identifying AI workloads in a sample application.

---

### 1. What is the main purpose of exploring a sample AI application?

A. To connect AI concepts to practical features  
B. To avoid learning AI terminology  
C. To replace all assessments  
D. To configure network hardware  

**Answer:** A  
**Rationale:** The exercise helps learners see how AI workloads appear in application features.

---

### 2. A sample app answers user questions in a chat interface. Which workload is most likely involved?

A. Generative AI and agents  
B. Disk formatting  
C. Manual filing  
D. Network cabling  

**Answer:** A  
**Rationale:** Chat responses commonly involve generative AI and may involve agents.

---

### 3. A sample app accepts spoken questions. Which workload is involved?

A. Speech  
B. Computer vision  
C. File compression  
D. Manual data entry  

**Answer:** A  
**Rationale:** Spoken input uses AI speech capabilities.

---

### 4. A sample app describes an uploaded image. Which workload is involved?

A. Computer vision  
B. Text-to-speech only  
C. Calendar scheduling  
D. File encryption  

**Answer:** A  
**Rationale:** Describing image content requires visual analysis.

---

### 5. A sample app extracts a serial number from a photo. Which workload is involved?

A. Information extraction  
B. Speech synthesis  
C. Sentiment analysis only  
D. Responsible AI only  

**Answer:** A  
**Rationale:** Extracting a specific value from an image is an information extraction scenario.

---

### 6. Why should learners map features to workloads during the exercise?

A. It builds scenario-recognition skills for AI-901  
B. It prevents them from finishing the exercise  
C. It replaces all lesson content  
D. It makes the app slower  

**Answer:** A  
**Rationale:** AI-901 often asks learners to identify the right AI capability for a requirement.

---

### 7. True or False: A single AI application can combine multiple AI workloads.

**Answer:** True  
**Rationale:** Real AI solutions often combine language, speech, vision, extraction, and responsible AI safeguards.

---

### 8. True or False: Responsible AI should be ignored during hands-on exploration.

**Answer:** False  
**Rationale:** Learners should consider risks and safeguards even in sample applications.

---

### 9. True or False: Feature mapping means connecting an application feature to the AI capability that supports it.

**Answer:** True  
**Rationale:** Feature mapping helps learners explain which workload is being used.

---

### 10. True or False: Exercise units should only be completed passively.

**Answer:** False  
**Rationale:** Learners should actively observe, classify, and reflect on what the exercise demonstrates.
