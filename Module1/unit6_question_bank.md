# Module 1: Introduction to AI Concepts

## Unit 6 Question Bank: Information extraction

These questions are authored from the unit's Microsoft Learn concepts and rewritten as exam-style practice items. Answer choices are intentionally paraphrased instead of copied from the lesson text.

---

### 1. A finance team wants invoice numbers, vendor names, and totals from scanned invoices. Which workload applies?

A. OCR  
B. field extraction  
C. content understanding  
D. Information extraction from documents  

**Answer:** D  
**Rationale:** The requirement matches the unit capability named in the correct option.

---

### 2. What is the main purpose of "information extraction"?

A. finding specific values such as dates, totals, names, or invoice numbers  
B. analyzing documents, images, audio, or video to extract useful data  
C. using AI to pull structured information from unstructured or semi-structured content  
D. recognizing printed or handwritten text in images or scanned documents  

**Answer:** C  
**Rationale:** "information extraction" is a core idea in this unit, and the correct option paraphrases its role.

---

### 3. A system must read text from photos of signs before analysis. Which capability is needed first?

A. Object detection  
B. Optical character recognition  
C. information extraction  
D. Prompt engineering  

**Answer:** B  
**Rationale:** The correct option is the unit concept that best fits the stated requirement.

---

### 4. How is OCR different from field extraction?

A. OCR reads text from visual content; field extraction identifies specific values and maps them to fields.  
B. OCR generates voices, while field extraction detects sentiment.  
C. Field extraction only creates images.  
D. Both are only used for chat prompts.  

**Answer:** A  
**Rationale:** The unit separates recognizing text from identifying meaningful structured values.

---

### 5. What should a learner remember about "OCR"?

A. a configuration task for the interface rather than the AI behavior  
B. a rule-based process that does not use model reasoning  
C. an access-control setting rather than the concept being defined  
D. reading or identifying printed or handwritten text in images or scanned documents  

**Answer:** D  
**Rationale:** "OCR" is tested by recognizing its purpose, not by memorizing wording from the page.

---

### 6. Why is information extraction useful for business workflows?

A. It replaces the need to define what fields matter.  
B. It only changes the color of scanned pages.  
C. It turns content such as forms or documents into structured data that applications can use.  
D. It makes all data private automatically.  

**Answer:** C  
**Rationale:** The value is moving from raw content to usable structured information.

---

### 7. Which two are examples of extracted fields? Choose two.

A. Display refresh setting  
B. Voice pitch only  
C. Image background color preference  
D. Invoice total  
E. Customer address  

**Answer:** D, E  
**Rationale:** Totals and addresses are structured values that can be extracted from documents.

### 8. A business wants structured values from forms that have different layouts. Which capability should it choose?

A. Extracting named fields  
B. Speech synthesis  
C. Model evaluation  
D. Choose a model based on popularity instead of task fit.  

**Answer:** A  
**Rationale:** The scenario points to the unit capability described by the correct option.

### 9. Which three content types can information extraction apply to in this course? Choose three.

A. Network firewall rules only  
B. Form layout metadata only  
C. Display settings rather than extracted content  
D. Documents  
E. Images  
F. Audio or video  

**Answer:** D, E, F  
**Rationale:** The unit frames extraction across documents and broader multimodal content.

### 10. Which description best matches "field extraction"?

A. Use a visual analysis capability when the input is only text.  
B. Handle sensitive data without the required access controls.  
C. locating specific values such as dates, totals, names, or invoice numbers  
D. Rely on one successful trial instead of broader evaluation.  

**Answer:** C  
**Rationale:** The correct answer paraphrases how the unit explains "field extraction".
