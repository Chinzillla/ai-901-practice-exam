# Module 1: Introduction to AI Concepts

## Unit 2: Generative AI and Agents

### Unit Overview

This unit introduces foundational AI-901 concepts related to generative AI and AI agents. Learners explore how generative AI creates new content, how language models respond to prompts, how large language models and small language models differ, and how AI agents combine models, instructions, and tools to automate tasks.

### Learning Objectives

By the end of this unit, learners should be able to:

- Define generative AI and identify common content formats it can create.
- Explain how prompts are used to interact with generative AI models.
- Describe the role of language models in generating meaningful responses.
- Distinguish between large language models and small language models.
- Identify the three key elements of an AI agent: model, instructions, and tools.
- Differentiate between knowledge tools and action tools.
- Recognize common generative AI and agentic AI scenarios.

### Key Terms

| Term | Definition |
| --- | --- |
| Generative AI | A branch of AI that enables applications to generate new content, such as text, images, video, code, and other formats. |
| Prompt | A natural language question, statement, or instruction used to guide a generative AI model's response. |
| Language model | An AI model trained on large volumes of data to understand language patterns and generate meaningful text. |
| Large language model (LLM) | A powerful language model trained with large amounts of data and many variables. |
| Small language model (SLM) | A smaller model that can work well for focused scenarios or local applications. |
| AI agent | A software application built on generative AI that can reason, use tools, and respond to context to take action. |
| Instructions | A system prompt that defines an agent's role and behavior. |
| Knowledge tool | A tool that gives an agent access to information, such as a search engine or database. |
| Action tool | A tool that enables an agent to perform a task, such as sending email or updating a calendar. |

### Lesson Flow

#### 1. Introduction to Generative AI

Start by defining generative AI as a branch of AI that creates new content. Emphasize that content can include natural language dialogs, images, video, code, documents, and other formats.

Use the computing history chatbot scenario as an example. A user can ask questions about important people, technologies, or events in computing history, and the chat interface can generate original responses.

Key teaching point: generative AI is not limited to retrieving a fixed answer. It can generate new responses based on the user's prompt and the model's training.

#### 2. How Generative AI Works

Explain that generative AI applications are commonly powered by language models. These models are trained using huge volumes of data, often including documents from the Internet and other public sources.

Learners should understand that users interact with these models through prompts. A prompt is the input that starts or guides the model's response.

Key teaching point: language models learn semantic relationships between language elements. In simpler terms, they learn how words and ideas relate to one another, which helps them generate meaningful text.

#### 3. LLMs and SLMs

Introduce the difference between large language models and small language models.

LLMs are generally powerful and flexible. They can generalize well across many tasks, but they can also be more costly to train and use.

SLMs are smaller and often better suited for focused topic areas, local applications, agents running on devices, or scenarios where deployment size and cost are important.

Key teaching point: the best model choice depends on the use case. Bigger is not automatically better for every scenario.

#### 4. Introduction to AI Agents

Define AI agents as software applications built on generative AI that can reason over and generate natural language, automate tasks using tools, and respond to contextual conditions.

Explain the three key elements of an AI agent:

- Model: The agent's language understanding and reasoning capability.
- Instructions: The system prompt that defines the agent's role and behavior.
- Tools: Capabilities the agent uses to interact with information or perform actions.

#### 5. Agent Tools

Explain that tools help agents interact with the world beyond text generation.

Knowledge tools provide access to information. Examples include search engines, databases, and document repositories.

Action tools allow agents to perform tasks. Examples include sending emails, updating calendars, controlling devices, or making changes in another system.

Key teaching point: tools are what allow an agent to move from answering questions to assisting with tasks.

#### 6. Common Generative and Agentic AI Scenarios

Review common use cases:

- Creating chatbots that answer user questions or engage in conversation.
- Implementing AI assistants that help users automate tasks.
- Creating drafts of documents or other content for further development.
- Translating text between languages.
- Summarizing or explaining complex documents.

### Suggested Activities

#### Activity 1: Identify the AI Scenario

Present learners with short examples and ask them to identify whether each one is generative AI, an AI agent, or neither.

Example scenarios:

- A chatbot answers questions about computing history.
- An assistant sends an email after a user asks it to follow up with a customer.
- A spreadsheet sorts rows alphabetically.
- A tool summarizes a complex policy document.

#### Activity 2: Prompt Practice

Ask learners to write three prompts for a generative AI system:

- One prompt that asks a factual question.
- One prompt that asks for a summary.
- One prompt that asks for content creation, such as a document draft or explanation.

Discuss how prompt wording affects the kind of response the model generates.

#### Activity 3: Knowledge Tool or Action Tool

Give learners a list of tools and ask them to classify each as a knowledge tool or action tool.

Examples:

- Search engine
- Customer database lookup
- Send email
- Update calendar
- Retrieve a document
- Control a smart device

### Checks for Understanding

Use these questions during or after instruction:

1. What is generative AI used to create?
2. What is a prompt?
3. Why can a language model generate meaningful text?
4. How are LLMs and SLMs different?
5. What are the three key elements of an AI agent?
6. What is the difference between a knowledge tool and an action tool?
7. Why might an AI assistant need tools?

### Teaching Notes

- Reinforce that generative AI can create many content types, not just text.
- Clarify that prompts are natural language inputs, not code-only commands.
- Emphasize that agents extend generative AI by using tools and context to complete tasks.
- Use simple examples, such as chatbots, document summarization, translation, and digital assistants, to connect concepts to real-world AI-901 scenarios.
