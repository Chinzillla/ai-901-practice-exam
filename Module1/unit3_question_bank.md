### 1. A company wants to know whether product reviews are positive, negative, or neutral. Which capability should it use?

A. natural language processing  
B. language detection  
C. PII redaction  
D. Sentiment analysis  

**Answer:** D  
**Rationale:** The requirement matches the unit capability named in the correct option.

### 2. What is the main purpose of "natural language processing"?

A. classifying text as positive, negative, or neutral  
B. finding and removing private information such as names, addresses, and phone numbers  
C. AI techniques that help software analyze, interpret, and work with human language  
D. identifying the language or languages used in a document  

**Answer:** C  
**Rationale:** "natural language processing" is a core idea in this unit, and the correct option paraphrases its role.

### 4. How does key-term extraction differ from entity detection?

A. Key-term extraction finds important phrases; entity detection finds specific things like people, places, and organizations.  
B. Key-term extraction translates audio, while entity detection generates images.  
C. Entity detection is only used for sentiment scores.

### 5. What should a learner remember about "language detection"?

### 6. Why is summarization useful in text analysis?

A. It removes the need to classify or review text.

**Rationale:** Summarization is described as reducing text volume while keeping the main ideas.
