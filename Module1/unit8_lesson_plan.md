# Module 1: Introduction to AI Concepts

## Unit 8: Exercise - Explore AI Workloads

### Source

Microsoft Learn: [Exercise - Explore AI workloads](https://learn.microsoft.com/en-us/training/modules/get-started-ai-fundamentals/7b-exercise)

### Unit Overview

This exercise unit gives learners an opportunity to explore the AI workloads introduced in the module. The goal is to connect concepts to a sample application experience and reinforce how generative AI, language, speech, computer vision, information extraction, and responsible AI can appear together in an AI solution.

### Learning Objectives

By the end of this unit, learners should be able to:

- Recognize multiple AI workloads in a sample application.
- Explain how a single application can combine different AI capabilities.
- Identify which workload supports a given feature.
- Reflect on responsible AI considerations in a practical experience.

### Key Terms

| Term | Definition |
| --- | --- |
| AI workload | A category of AI capability used to solve a type of problem. |
| Sample application | An application used for exploration, demonstration, or learning. |
| Integrated AI solution | A solution that combines multiple AI capabilities to support a user experience. |

### Lesson Flow

#### 1. Set the Exercise Purpose

Explain that the exercise is not just a click-through activity. Learners should actively identify which AI workload supports each feature they observe.

#### 2. Connect Features to Workloads

As learners explore, ask them to map features to workloads:

- Chat or generated answers: generative AI and agents.
- Text analysis: natural language processing.
- Voice input or output: speech.
- Image upload or description: computer vision.
- Extracting values from documents or images: information extraction.
- Safeguards or access considerations: responsible AI.

#### 3. Debrief

After the exercise, review which workloads appeared and how they worked together.

### Suggested Activities

#### Activity 1: Workload Checklist

Learners complete a checklist of AI workloads they observe in the sample application.

#### Activity 2: Feature Mapping

Learners select three observed features and explain which AI capability supports each one.

#### Activity 3: Responsible AI Reflection

Learners identify one potential risk and one safeguard that would improve the application.

### Checks for Understanding

1. Which workloads did the sample application demonstrate?
2. How can one application use multiple AI capabilities?
3. Which feature required speech?
4. Which feature required computer vision?
5. What responsible AI consideration did you notice?
