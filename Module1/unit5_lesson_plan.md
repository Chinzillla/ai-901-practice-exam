# Module 1: Introduction to AI Concepts

## Unit 5: Computer Vision

### Source

Microsoft Learn: [Computer vision](https://learn.microsoft.com/en-us/training/modules/get-started-ai-fundamentals/3-computer-vision)

### Unit Overview

This unit introduces computer vision as the area of AI focused on analyzing visual input, including photographs, videos, and live camera feeds. Learners explore common computer vision techniques such as image classification, object detection, semantic segmentation, and multimodal image description.

### Learning Objectives

By the end of this unit, learners should be able to:

- Define computer vision.
- Identify common sources of visual input.
- Explain how image classification differs from object detection.
- Describe semantic segmentation at a high level.
- Recognize how multimodal models combine visual and text information.
- Choose computer vision capabilities for scenario-based requirements.

### Key Terms

| Term | Definition |
| --- | --- |
| Computer vision | AI capabilities that analyze visual input such as images, video, and camera feeds. |
| Image classification | Predicting the main label or category for an image. |
| Object detection | Identifying the location of specific objects in an image. |
| Semantic segmentation | Identifying the individual pixels that belong to a specific object or area. |
| Multimodal model | A model that can combine information from more than one type of input, such as images and text. |
| Visual search | Searching for information by using an image or visual features. |

### Lesson Flow

#### 1. Introduce Computer Vision

Explain that computer vision helps AI systems analyze visual information. Examples include identifying objects in a photo, generating captions for images, monitoring video feeds, and helping robots interpret their surroundings.

#### 2. How Computer Vision Models Learn

Explain that computer vision models are trained with many images. The training data helps the model learn patterns in visual features so it can analyze new images.

Keep this conceptual. Learners do not need to understand model training mathematics in this unit.

#### 3. Image Classification

Define image classification as predicting the main subject or label for an image. For example, a model might classify an uploaded image as a vintage computer, keyboard, monitor, or circuit board.

Key teaching point: image classification answers "What is this image mainly of?"

#### 4. Object Detection

Define object detection as identifying where specific objects are located in an image. It is useful when the system must find multiple objects or locate a target object.

Key teaching point: object detection answers "What objects are present, and where are they?"

#### 5. Semantic Segmentation

Explain semantic segmentation as a more detailed form of visual analysis. Instead of drawing a box around an object, segmentation identifies the pixels that belong to the object.

Key teaching point: segmentation is useful when precise boundaries matter.

#### 6. Multimodal Image Understanding

Explain that multimodal models can combine visual features with text descriptions. This allows an AI system to generate richer descriptions of images.

#### 7. Computer Vision Scenarios

Review common uses:

- AI agents interpreting visual input.
- Auto-captioning and tag generation.
- Visual search.
- Retail stock monitoring or checkout item identification.
- Security video monitoring.
- Facial recognition.
- Robotics and self-driving vehicles.

### Suggested Activities

#### Activity 1: Pick the Vision Task

Give learners scenarios and ask whether the best fit is image classification, object detection, semantic segmentation, or multimodal description.

#### Activity 2: Describe the Image Pipeline

Ask learners to describe how an uploaded photo could be analyzed, labeled, and captioned by a computer vision system.

#### Activity 3: Scenario Sorting

Have learners sort examples into vision scenarios: retail, security, accessibility, robotics, search, or content organization.

### Checks for Understanding

1. What is computer vision?
2. What is the difference between image classification and object detection?
3. When might semantic segmentation be useful?
4. What does a multimodal model combine?
5. Which computer vision capability would identify where objects appear in an image?

### Teaching Notes

- Reinforce the difference between classifying an image and locating objects.
- Use simple visual examples: one image label, boxes around objects, and pixel-level regions.
- Connect computer vision to AI-901 scenarios where a requirement describes images, videos, camera feeds, or facial recognition.
