# Module 1: Introduction to AI Concepts

## Unit 10: Summary

### Source

Microsoft Learn: [Summary](https://learn.microsoft.com/en-us/training/modules/get-started-ai-fundamentals/9-summary)

### Unit Overview

This unit summarizes the core AI capabilities introduced in Module 1. Learners review generative AI and agents, natural language processing, speech, computer vision, information extraction, and responsible AI.

### Learning Objectives

By the end of this unit, learners should be able to:

- Summarize the AI workloads covered in Module 1.
- Explain the purpose of each workload in simple terms.
- Match each workload to a common scenario.
- Identify topics that need further review before continuing.

### Key Terms

| Term | Definition |
| --- | --- |
| Generative AI | AI that creates new content from prompts. |
| Agent | An AI application that can reason, use tools, and perform tasks for users. |
| NLP | AI techniques for understanding and analyzing language. |
| Speech AI | AI capabilities for spoken input and output. |
| Computer vision | AI for analyzing images, videos, and camera feeds. |
| Information extraction | AI for finding useful values and insights in unstructured sources. |
| Responsible AI | Principles and safeguards for trustworthy AI use. |

### Lesson Flow

#### 1. Review the Workloads

Summarize each workload in one or two sentences and connect it to a concrete example.

#### 2. Compare Similar Capabilities

Review common comparisons:

- Speech-to-text versus text-to-speech.
- Image classification versus object detection.
- OCR versus field extraction.
- Generative AI versus specialist text analysis.

#### 3. Prepare for the Next Module

Explain that future modules go deeper into these areas, starting with generative AI and agents.

### Suggested Activities

#### Activity 1: Workload Flashcards

Learners create flashcards with one workload on one side and its use cases on the other.

#### Activity 2: Scenario Exit Ticket

Learners write three scenarios and identify the best AI workload for each.

#### Activity 3: Review Plan

Learners identify two strong areas and one area to review before continuing.

### Checks for Understanding

1. Which workload generates new content?
2. Which workload analyzes language?
3. Which workload works with spoken audio?
4. Which workload analyzes images?
5. Which workload extracts values from unstructured data?
6. Which concept helps reduce harm and support trust?
