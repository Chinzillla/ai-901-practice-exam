# Module 1: Introduction to AI Concepts

## Unit 4 Question Bank: Speech

These questions are authored from the unit's Microsoft Learn concepts and rewritten as exam-style practice items. Answer choices are intentionally paraphrased instead of copied from the lesson text.

---

### 1. A meeting app creates written transcripts while people talk. Which capability is required?

A. speech synthesis  
B. speech translation  
C. voice-enabled assistant  
D. Speech recognition  

**Answer:** D  
**Rationale:** The requirement matches the unit capability named in the correct option.

---

### 2. What is the main purpose of "speech recognition"?

A. recognizing speech in one language and producing translated output  
B. an app that uses speech input or output to create a conversational experience  
C. converting spoken audio into text  
D. converting text into spoken audio  

**Answer:** C  
**Rationale:** "speech recognition" is a core idea in this unit, and the correct option paraphrases its role.

---

### 3. A navigation app reads directions aloud to a driver. Which capability is being used?

A. Information extraction  
B. Text-to-speech generation  
C. Prompt engineering  
D. Object detection  

**Answer:** B  
**Rationale:** The correct option is the unit concept that best fits the stated requirement.

---

### 4. What is the difference between speech recognition and speech synthesis?

A. Recognition turns audio into text; synthesis turns text into audio.  
B. Recognition creates images, while synthesis extracts tables.  
C. Recognition is only for translation, while synthesis is only for security.  
D. Both only detect whether text is positive or negative.  

**Answer:** A  
**Rationale:** The unit distinguishes speech-to-text from text-to-speech.

---

### 5. What should a learner remember about "speech synthesis"?

A. a configuration task for the interface rather than the AI behavior  
B. a rule-based process that does not use model reasoning  
C. an access-control setting rather than the concept being defined  
D. turning text into spoken audio  

**Answer:** D  
**Rationale:** "speech synthesis" is tested by recognizing its purpose, not by memorizing wording from the page.

---

### 6. Why are speech capabilities useful in accessibility scenarios?

A. They guarantee all translations are legally certified.  
B. They replace every visual interface with image generation.  
C. They let users interact through spoken input or audio output when text or screens are difficult to use.  
D. They remove the need for privacy review.  

**Answer:** C  
**Rationale:** Speech supports more natural and accessible interaction patterns.

---

### 7. Which two are examples of speech-enabled solutions? Choose two.

A. Detecting faces in a photo  
B. Extracting fields from a form  
C. Classifying a review as negative  
D. Dictating notes into an app  
E. Having a system read text aloud  

**Answer:** D, E  
**Rationale:** Dictation uses recognition, and read-aloud uses synthesis.

---

### 8. A traveler speaks English and the app outputs the same message in Japanese. What capability is involved?

A. Spoken-language translation  
B. Model evaluation  
C. Choose a model based on popularity instead of task fit.  
D. Rely on one successful trial instead of broader evaluation.  

**Answer:** A  
**Rationale:** The scenario points to the unit capability described by the correct option.

---

### 9. Which three concerns matter when designing a speech solution? Choose three.

A. Image resolution  
B. Bounding box labels  
C. Database table indexes only  
D. Audio quality  
E. Language support  
F. Latency for real-time conversations  

**Answer:** D, E, F  
**Rationale:** Speech systems depend on audio conditions, supported languages, and response timing.

---

### 10. Which description best matches "speech translation"?

A. Handle sensitive data without the required access controls.  
B. Leave the model without clear task instructions.  
C. reading or identifying speech in one language and producing translated output  
D. Use a visual analysis capability when the input is only text.  

**Answer:** C  
**Rationale:** The correct answer paraphrases how the unit explains "speech translation".
