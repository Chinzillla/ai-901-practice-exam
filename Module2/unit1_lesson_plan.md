# Module 2: Introduction to Generative AI and Agents

## Unit 1: Introduction

### Source

Microsoft Learn: [Introduction](https://learn.microsoft.com/en-us/training/modules/fundamentals-generative-ai/1-introduction)

### Unit Overview

This unit introduces generative AI as a fast-growing field that can produce human-like original content, including text, code, and creative writing. Learners should understand that generative AI may feel surprising, but it is based on mathematical, statistical, data science, and machine learning techniques rather than magic.

### Learning Objectives

By the end of this unit, learners should be able to:

- Describe why generative AI has become widely visible.
- Recognize examples of generated content.
- Explain that generative AI is based on machine learning techniques.
- Describe why understanding core concepts helps learners use generative AI responsibly and effectively.

### Key Terms

| Term | Definition |
| --- | --- |
| Generative AI | AI that can create original content such as text, code, images, or other outputs. |
| Machine learning | Techniques that allow models to learn patterns from data. |
| Human-like content | Output that resembles content a person might write or create. |
| AI agent | A generative AI-based application that can reason, use tools, and act on behalf of a user. |

### Lesson Flow

#### 1. Why Generative AI Feels Different

Explain that many people encounter generative AI because it can produce fluent and human-like content. It can write prose, summarize information, answer questions, and generate code.

#### 2. No Magic, Just Models

Clarify that generative AI is not magic. It uses techniques developed through statistics, data science, and machine learning to learn patterns and generate outputs.

#### 3. Why Concepts Matter

Explain that learners do not need to master the underlying math for AI-901, but they should understand core concepts such as language models, prompts, and agents.

#### 4. Preview the Module

Preview the major topics:

- Large language models.
- Prompts.
- AI agents.
- Exercises and assessment review.

### Suggested Activities

#### Activity 1: Content Examples

Ask learners to list examples of content generative AI might create.

#### Activity 2: Myth or Concept

Ask learners to identify whether statements describe a realistic AI concept or an exaggerated claim.

### Checks for Understanding

1. What kinds of content can generative AI produce?
2. Why is generative AI not magic?
3. Which concepts will this module explore?
4. Why should learners understand prompts and agents?
