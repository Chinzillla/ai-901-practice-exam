### 1. A support assistant must search product documentation before answering. What agent component enables this?

A. AI agent  
B. instructions  
C. agent orchestration  
D. A knowledge tool  

**Answer:** D  
**Rationale:** This scenario matches AI agents as described in the unit.

### 2. What is the main purpose of "AI agent"?

A. the system guidance that defines how the agent should behave  
B. the process of deciding which steps or tools are needed to complete a task  
C. a generative AI application that can reason, follow instructions, and use tools  
D. a capability the agent can call to retrieve information or perform an action  

**Answer:** C  
**Rationale:** "AI agent" is a core idea in this unit, and the correct option paraphrases its role.

### 3. An assistant books a meeting after checking availability. What makes this an agentic scenario?

**Answer:** B  
**Rationale:** The correct option is the unit concept that best fits the stated requirement.

### 4. How is an agent different from a basic generative AI chat experience?

A. An agent can use instructions and tools to plan or take actions beyond generating text.  
B. An agent cannot use language models.  
C. A basic chatbot always has more tools than an agent.  
D. Agents are only used for image classification.  

**Answer:** A  
**Rationale:** The unit presents tools and instructions as what extend agents beyond simple chat.

### 5. What should a learner remember about "tool"?

A. a process for changing the visual theme of the development portal  
B. a manual-only activity that does not use an AI model  
C. a security shortcut that removes the need for access controls  
D. a capability the agent can call to retrieve information or perform an action as described in this unit  

**Answer:** D  
**Rationale:** "tool" is tested by recognizing its purpose, not by memorizing wording from the page.

---

### 6. Why do agents need tools?

A. Tools are only decorative labels.  
B. Tools replace all need for user consent.  
C. Tools let the agent access current information or perform actions outside the model itself.  
D. Tools prevent the agent from using instructions.  

**Answer:** C  
**Rationale:** Tools connect the agent to data sources and external actions.

---

### 7. Which two are examples of agent tools? Choose two.

A. Changing the learner's exam score manually  
B. Ignoring all context  
C. Turning every prompt into an image  
D. Searching a database  
E. Updating a calendar  

**Answer:** D, E  
**Rationale:** Knowledge tools retrieve information; action tools do work in external systems.
