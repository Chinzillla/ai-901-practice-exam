# Module 2: Introduction to Generative AI and Agents

## Unit 7 Question Bank: Summary

These questions provide cumulative review for Module 2.

---

### 1. Which technology is central to many generative AI applications?

A. Large language model  
B. Network cable  
C. Printer queue  
D. Disk cleaner  

**Answer:** A  
**Rationale:** LLMs power many generative AI language experiences.

---

### 2. What guides a model response?

A. Prompt  
B. Monitor brightness  
C. Keyboard color  
D. File size only  

**Answer:** A  
**Rationale:** Prompts provide input and instructions for a response.

---

### 3. What helps ground model responses in specific source content?

A. Retrieval augmented generation  
B. Object detection  
C. Speech synthesis  
D. Disk formatting  

**Answer:** A  
**Rationale:** RAG retrieves relevant information and includes it as context.

---

### 4. What lets an agent interact with external systems?

A. Tools  
B. Token IDs only  
C. Static text  
D. Printed instructions only  

**Answer:** A  
**Rationale:** Tools allow agents to retrieve information or perform actions.

---

### 5. Which item defines an agent's role and behavior?

A. Instructions  
B. Monitor settings  
C. Camera resolution  
D. Cable length  

**Answer:** A  
**Rationale:** Instructions act like the agent's job description.

---

### 6. True or False: Prompt quality can affect response quality.

**Answer:** True  
**Rationale:** Clear, specific prompts usually produce more useful results.

---

### 7. True or False: Agents can perform tasks by using tools.

**Answer:** True  
**Rationale:** Tool use is what lets agents do more than answer questions.

---

### 8. True or False: RAG means the model ignores retrieved documents.

**Answer:** False  
**Rationale:** RAG uses retrieved documents to ground the response.

---

### 9. True or False: Embeddings help represent semantic meaning.

**Answer:** True  
**Rationale:** Embeddings capture relationships between tokens in vector form.

---

### 10. True or False: Multi-agent systems can divide work among specialized agents.

**Answer:** True  
**Rationale:** Multi-agent systems can coordinate agents with different roles.
