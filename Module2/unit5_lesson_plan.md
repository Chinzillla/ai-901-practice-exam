# Module 2: Introduction to Generative AI and Agents

## Unit 5: Exercise - Explore Generative AI

### Source

Microsoft Learn: [Exercise - Explore generative AI](https://learn.microsoft.com/en-us/training/modules/fundamentals-generative-ai/7a-exercise)

### Unit Overview

This exercise unit gives learners practice interacting with a generative AI model in a chat playground. Learners observe the effects of system prompts, model parameters, and grounding with data.

### Learning Objectives

By the end of this unit, learners should be able to:

- Explore how prompts influence model responses.
- Observe how system prompts shape model behavior.
- Recognize that model parameters can affect output.
- Explain how grounding with data can improve relevance.

### Key Terms

| Term | Definition |
| --- | --- |
| Chat playground | An interactive environment for testing prompts and model responses. |
| System prompt | Instructions that define model behavior and constraints. |
| Model parameter | A configurable setting that can affect model output. |
| Grounding | Supplying relevant data to help the model respond in context. |

### Lesson Flow

#### 1. Exercise Orientation

Explain that learners should observe how changes affect model behavior, not just submit prompts.

#### 2. System Prompt Effects

Ask learners to compare model responses before and after changing role, tone, or constraints.

#### 3. Model Parameters

Discuss that model settings can affect behavior such as response creativity or consistency. Keep the explanation high level.

#### 4. Grounding with Data

Explain that grounding helps the model answer using relevant information instead of generic knowledge.

### Suggested Activities

#### Activity 1: Prompt Comparison

Learners submit a vague prompt and then a clearer prompt. They compare the output quality.

#### Activity 2: System Prompt Test

Learners change the assistant's role or tone and observe differences.

#### Activity 3: Grounded Response Reflection

Learners explain how grounding data changed or improved a response.

### Checks for Understanding

1. What is the purpose of a chat playground?
2. How can a system prompt affect output?
3. Why might grounding data improve a response?
4. What should learners observe during the exercise?
