## Unit 3 Question Bank: Prompts

These questions are authored from the unit's Microsoft Learn concepts and rewritten as exam-style practice items. Answer choices are intentionally paraphrased instead of copied from the lesson text.

### 1. A teacher wants a model to answer as a patient beginner-friendly tutor. Where should that behavior be defined?

A. user prompt  
B. grounding  
C. Prompt engineering  
D. In a system prompt  

**Answer:** D  
**Rationale:** This scenario matches Prompts as described in the unit.

### 2. What is the main purpose of "prompt"?

A. the task request or question submitted by the user  
B. providing context or source information to guide the model's response  
C. the instruction or question given to a generative AI model  
D. instructions that define the model or agent's role, tone, and boundaries  

**Answer:** C  
**Rationale:** "prompt" is a core idea in this unit, and the correct option paraphrases its role.

### 3. A user asks a model to summarize a paragraph in three bullets. What is that request?

A. Speech synthesis  
B. A user prompt  
C. system prompt  
D. Object detection  

**Answer:** B  
**Rationale:** The correct option is the unit concept that best fits the stated requirement.

### 4. How is a system prompt different from a user prompt?

A. A system prompt sets behavior and constraints; a user prompt asks for the current task.  
B. A user prompt controls hidden safety filters, while a system prompt is ignored.  
C. System prompts are only for images, while user prompts are only for speech.  
D. They are the same thing and always have the same priority.  

**Answer:** A  
**Rationale:** The unit separates role-setting instructions from user task requests.

### 5. What should a learner remember about "system prompt"?

A. a process for changing the visual theme of the development portal  
B. a manual-only activity that does not use an AI model  
C. a security shortcut that removes the need for access controls  
D. instructions that define the model or agent's role, tone, and boundaries as described in this unit  

**Answer:** D  
**Rationale:** "system prompt" is tested by recognizing its purpose, not by memorizing wording from the page.

### 6. Why should prompts include clear output expectations?

A. Output format matters only for computer vision.  
B. Prompts cannot influence model responses.  
C. Clear format and task details help the model produce a more useful response.  
D. Vague prompts always produce more accurate results.  

**Answer:** C  
**Rationale:** Effective prompting improves relevance and format control.

---

### 7. Which two details improve a prompt? Choose two.

A. Contradictory instructions  
B. No context for the request  
C. A command to ignore safety limits  
D. A clear task description  
E. The desired response format  

**Answer:** D, E  
**Rationale:** Specific instructions and desired format make model output easier to use.

### 8. A team adds a policy excerpt to the prompt so answers stay based on approved content. What technique is this?

A. Grounding the prompt with context  
B. Information extraction  
C. Model evaluation  
D. Choose a model only because it has the largest name recognition.  

**Answer:** A  
**Rationale:** The scenario points to the unit capability described by the correct option.

---

### 9. Which three elements can be included in effective prompts? Choose three.

A. A requirement to hallucinate  
B. Unrelated image labels  
C. Hidden network credentials  
D. Role or tone guidance  
E. Relevant context  
F. Output constraints  

**Answer:** D, E, F  
**Rationale:** Prompt quality depends on clear role, context, and constraints.
