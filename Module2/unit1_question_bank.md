# Module 2: Introduction to Generative AI and Agents

## Unit 1 Question Bank: Introduction

These questions reinforce introductory concepts about generative AI and why it matters.

---

### 1. What is generative AI known for?

A. Creating original content such as text or code  
B. Replacing all computer hardware  
C. Only storing data in tables  
D. Preventing users from asking questions  

**Answer:** A  
**Rationale:** Generative AI creates new content based on patterns learned from data.

---

### 2. Which output is an example of generative AI?

A. A newly written paragraph generated from a prompt  
B. A disconnected network cable  
C. A manually sorted folder  
D. A blank hard drive  

**Answer:** A  
**Rationale:** Generating original text from a prompt is a generative AI capability.

---

### 3. Why can generative AI seem surprising to new users?

A. It can produce human-like original content  
B. It only works when offline  
C. It cannot respond to language  
D. It never uses machine learning  

**Answer:** A  
**Rationale:** Fluent text, code, and creative outputs can make generative AI feel unusually capable.

---

### 4. Generative AI is best understood as being based on:

A. Mathematical, statistical, data science, and machine learning techniques  
B. Magic or wizardry  
C. Manual typing of every possible answer  
D. Hardware replacement only  

**Answer:** A  
**Rationale:** Generative AI is based on learned patterns and model techniques, not magic.

---

### 5. Which concept is central to understanding generative AI?

A. Large language models  
B. Printer toner  
C. Network cables  
D. Disk cleanup only  

**Answer:** A  
**Rationale:** LLMs are a major technology behind many generative AI applications.

---

### 6. Which concept describes input given to a model to get a response?

A. Prompt  
B. Router  
C. Monitor  
D. Cable  

**Answer:** A  
**Rationale:** A prompt is the input used to guide a generative AI model's response.

---

### 7. What can AI agents do beyond simply answering questions?

A. Use tools and act on behalf of users  
B. Replace every application automatically  
C. Work only without instructions  
D. Avoid all context  

**Answer:** A  
**Rationale:** Agents can reason, use tools, and perform tasks.

---

### 8. Why is it useful to study generative AI concepts before implementation?

A. It helps learners understand what the technology can do and how to use it appropriately  
B. It eliminates all risk  
C. It makes code unnecessary in every scenario  
D. It prevents model deployment  

**Answer:** A  
**Rationale:** Conceptual understanding helps learners evaluate capabilities, limitations, and scenarios.

---

### 9. True or False: Generative AI can create computer code.

**Answer:** True  
**Rationale:** Generative AI can produce code as well as text and other content.

---

### 10. True or False: Generative AI is based on magic rather than machine learning techniques.

**Answer:** False  
**Rationale:** Generative AI is based on mathematical and machine learning techniques.

---

### 11. True or False: Understanding prompts is useful when working with generative AI.

**Answer:** True  
**Rationale:** Prompts guide model responses and strongly affect output quality.

---

### 12. True or False: AI agents are unrelated to generative AI.

**Answer:** False  
**Rationale:** AI agents are built on generative AI capabilities.
