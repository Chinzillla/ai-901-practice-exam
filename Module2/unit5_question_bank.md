# Module 2: Introduction to Generative AI and Agents

## Unit 5 Question Bank: Exercise - Explore Generative AI

These questions reinforce the exercise concepts of prompts, system prompts, model parameters, and grounding.

---

### 1. What is the purpose of using a chat playground?

A. To test prompts and observe model responses  
B. To replace all learning objectives  
C. To configure network switches  
D. To delete model settings  

**Answer:** A  
**Rationale:** A chat playground lets learners experiment with model interactions.

---

### 2. What should learners observe when changing a system prompt?

A. How role, tone, or constraints affect responses  
B. Whether the computer monitor changes color  
C. Whether files are compressed  
D. Whether the model stops accepting input permanently  

**Answer:** A  
**Rationale:** System prompts guide model behavior and response style.

---

### 3. What does grounding with data help the model do?

A. Produce responses based on relevant provided information  
B. Ignore all context  
C. Generate only random output  
D. Convert all text into images  

**Answer:** A  
**Rationale:** Grounding gives the model context for more relevant answers.

---

### 4. Why compare a vague prompt with a clear prompt?

A. To see how prompt quality affects output quality  
B. To prove prompts do not matter  
C. To remove all model context  
D. To avoid learning from the exercise  

**Answer:** A  
**Rationale:** Prompt clarity can significantly affect response usefulness.

---

### 5. What can model parameters affect?

A. The behavior or style of model output  
B. The physical size of the keyboard  
C. The user's chair height  
D. The network cable color  

**Answer:** A  
**Rationale:** Parameters can influence how a model generates responses.

---

### 6. True or False: Exercise observations should connect back to prompt and grounding concepts.

**Answer:** True  
**Rationale:** The exercise helps learners see these concepts in practice.

---

### 7. True or False: A system prompt cannot affect a model response.

**Answer:** False  
**Rationale:** System prompts are designed to affect model behavior and constraints.

---

### 8. True or False: Grounding can help avoid generic answers when specific source data is available.

**Answer:** True  
**Rationale:** Grounding supplies relevant information for the model to use.

---

### 9. True or False: Learners should only submit prompts without reflecting on the results.

**Answer:** False  
**Rationale:** Reflection helps learners understand cause and effect in model behavior.

---

### 10. True or False: A chat playground can help learners experiment safely before building an app.

**Answer:** True  
**Rationale:** Playgrounds are useful for exploration and prompt testing.
