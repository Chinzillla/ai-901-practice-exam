# Module 2: Introduction to Generative AI and Agents

## Unit 3: Prompts

### Source

Microsoft Learn: [Prompts](https://learn.microsoft.com/en-us/training/modules/fundamentals-generative-ai/6-writing-prompts)

### Unit Overview

This unit explains prompts as the inputs used to get responses from large language models. Learners compare system prompts and user prompts, explore conversation history, learn how retrieval augmented generation adds grounding context, and practice prompt-quality techniques.

### Learning Objectives

By the end of this unit, learners should be able to:

- Define prompts and completions.
- Distinguish between system prompts and user prompts.
- Explain how conversation history supports continuity.
- Describe retrieval augmented generation at a high level.
- Apply prompt-writing tips such as adding context, examples, and structure.

### Key Terms

| Term | Definition |
| --- | --- |
| Prompt | Input given to an LLM to get a response. |
| Completion | The model's generated response to a prompt. |
| System prompt | Instructions that set model behavior, tone, role, or constraints. |
| User prompt | A user's question, command, or instruction to the model. |
| Conversation history | Prior messages included as context for later responses. |
| Retrieval augmented generation (RAG) | Retrieving relevant information and adding it to the prompt so the model can produce grounded responses. |
| Grounding | Providing relevant source information to help the model answer in context. |

### Lesson Flow

#### 1. Define Prompts and Completions

Explain that a prompt is the input and the completion is the response. Prompts can be questions, instructions, or conversational messages.

#### 2. System Prompts

Explain that system prompts define role, tone, behavior, and constraints. They are usually set by the application.

#### 3. User Prompts

Explain that user prompts ask for a specific response or task. Good user prompts are clear, specific, contextual, and structured.

#### 4. Conversation History

Explain that generative AI apps may include previous messages so the model can understand follow-up questions.

#### 5. Retrieval Augmented Generation

Explain RAG as retrieving information from sources such as documents or emails and adding it to the prompt. This grounds the response in relevant data.

#### 6. Prompt Quality Tips

Review tips:

- Be clear and specific.
- Add context.
- Provide examples.
- Ask for structure.

### Suggested Activities

#### Activity 1: Improve the Prompt

Learners revise vague prompts into clear, specific prompts with context and output format.

#### Activity 2: System or User Prompt

Learners classify examples as system prompts or user prompts.

#### Activity 3: RAG Scenario

Learners identify when an application should retrieve policy documents before asking a model to answer.
