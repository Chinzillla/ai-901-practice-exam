# Module 2: Introduction to Generative AI and Agents

## Unit 6 Question Bank: Module Assessment Prep

These original questions prepare learners for Module 2 assessment concepts.

---

### 1. Which concept breaks text into smaller units for model processing?

A. Tokenization  
B. Speech synthesis  
C. Object detection  
D. Data backup  

**Answer:** A  
**Rationale:** Tokenization breaks text into tokens.

---

### 2. Which concept represents tokens as vectors that capture meaning?

A. Embeddings  
B. Calendar events  
C. File folders  
D. Network routes  

**Answer:** A  
**Rationale:** Embeddings are vector-based semantic representations.

---

### 3. Which technique helps a model evaluate relationships between tokens?

A. Attention  
B. OCR  
C. Speech-to-text  
D. Visual search  

**Answer:** A  
**Rationale:** Attention helps models understand which tokens influence one another.

---

### 4. Which prompt type defines the assistant's role and behavior?

A. System prompt  
B. User prompt only  
C. Completion  
D. Token ID  

**Answer:** A  
**Rationale:** System prompts set behavior, tone, role, and constraints.

---

### 5. Which technique retrieves documents and adds them to a prompt for context?

A. Retrieval augmented generation  
B. Object detection  
C. Speech synthesis  
D. Semantic segmentation  

**Answer:** A  
**Rationale:** RAG grounds a model response with retrieved information.

---

### 6. Which three elements are key to an AI agent?

A. Model, instructions, and tools  
B. Screen, cable, and printer  
C. Row, column, and cell  
D. Folder, shortcut, and icon  

**Answer:** A  
**Rationale:** Agents combine a language model, behavior instructions, and tools.

---

### 7. A tool searches a knowledge base for facts. What kind of tool is it?

A. Knowledge tool  
B. Action tool only  
C. Drawing tool  
D. Hardware tool  

**Answer:** A  
**Rationale:** Knowledge tools retrieve information.

---

### 8. A tool sends an email after user approval. What kind of tool is it?

A. Action tool  
B. Knowledge tool only  
C. Embedding  
D. Token  

**Answer:** A  
**Rationale:** Sending email performs an action in an external system.

---

### 9. True or False: A completion is generated in response to a prompt.

**Answer:** True  
**Rationale:** The completion is the model's output.

---

### 10. True or False: RAG is useful when a model should answer from organization-specific documents.

**Answer:** True  
**Rationale:** RAG can retrieve and include relevant organizational content.

---

### 11. True or False: AI agents cannot use tools.

**Answer:** False  
**Rationale:** Tool use is a core agent capability.

---

### 12. True or False: Multi-agent systems can include agents with different specialties.

**Answer:** True  
**Rationale:** Multi-agent systems can divide work across specialized agents.
