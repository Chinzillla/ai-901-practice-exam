# Module 2: Introduction to Generative AI and Agents

## Unit 4: AI Agents

### Source

Microsoft Learn: [AI agents](https://learn.microsoft.com/en-us/training/modules/fundamentals-generative-ai/7-agents)

### Unit Overview

This unit explains AI agents as generative AI-powered applications that can reason, use tools, and take action based on context. Learners explore the core components of an agent, the difference between knowledge tools and action tools, and the idea of multi-agent systems.

### Learning Objectives

By the end of this unit, learners should be able to:

- Define an AI agent.
- Identify the three key components of an AI agent.
- Explain the purpose of the model, instructions, and tools.
- Distinguish between knowledge tools and action tools.
- Describe multi-agent systems at a high level.
- Recognize agentic AI scenarios.

### Key Terms

| Term | Definition |
| --- | --- |
| AI agent | A generative AI-based application that can reason, use tools, and act on behalf of a user. |
| Large language model | The agent's language understanding and reasoning component. |
| Instructions | The system prompt that defines the agent's role and behavior. |
| Tool | A capability that lets the agent access information or perform actions. |
| Knowledge tool | A tool that provides access to information, such as a search engine or database. |
| Action tool | A tool that performs tasks, such as sending email or updating a calendar. |
| Multi-agent system | A system where multiple agents collaborate, often with different specialties. |

### Lesson Flow

#### 1. Define AI Agents

Explain that agents go beyond answering questions. They can reason, use tools, and perform tasks for users.

#### 2. Components of an Agent

Review the three key elements:

- Model: the reasoning and language engine.
- Instructions: the role and behavior definition.
- Tools: capabilities for information access or action.

#### 3. Knowledge Tools and Action Tools

Explain that knowledge tools retrieve information, while action tools perform tasks. Use examples such as search, databases, email, calendars, and device control.

#### 4. Multi-Agent Systems

Explain that multiple agents can collaborate. One agent might gather data, another analyze it, and another take action.

#### 5. Agentic AI Scenarios

Discuss digital assistants, workflow automation, research assistants, scheduling agents, and support agents.

### Suggested Activities

#### Activity 1: Build an Agent Card

Learners define an agent's role, instructions, knowledge tools, and action tools.

#### Activity 2: Tool Sorting

Learners classify tools as knowledge tools or action tools.

#### Activity 3: Multi-Agent Workflow

Learners design a simple workflow with two or three specialized agents.

### Checks for Understanding

3. What is the difference between knowledge tools and action tools?
4. What is a multi-agent system?
5. Why are instructions important for an agent?
