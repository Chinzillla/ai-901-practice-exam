# Module 2: Introduction to Generative AI and Agents

## Unit 7: Summary

### Source

Microsoft Learn: [Summary](https://learn.microsoft.com/en-us/training/modules/fundamentals-generative-ai/9-summary)

### Unit Overview

This unit summarizes how generative AI uses language models, prompts, and agents. Learners review how LLMs generate responses, how prompts guide responses, and how agents use tools to help users find information and complete tasks.

### Learning Objectives

By the end of this unit, learners should be able to:

- Summarize how LLMs support generative AI.
- Explain how prompts influence model responses.
- Describe how grounding can improve relevance.
- Explain how agents act as digital assistants.
- Identify areas for review before continuing.

### Key Terms

| Term | Definition |
| --- | --- |
| Generative AI | AI that creates new content from prompts. |
| LLM | A model that learns language relationships and generates responses. |
| Prompt | Input that guides a model's response. |
| RAG | A grounding approach that retrieves relevant information for the prompt. |
| AI agent | A generative AI application that can use tools to complete tasks. |

### Lesson Flow

#### 1. Review LLMs

Summarize tokenization, embeddings, attention, and next-token prediction in learner-friendly terms.

#### 2. Review Prompts

Review system prompts, user prompts, conversation history, and clear prompt design.

#### 3. Review Agents

Summarize how agents combine a model, instructions, and tools.

#### 4. Prepare for Future Modules

Explain that later modules and learning paths go deeper into implementation using Microsoft Foundry.

### Suggested Activities

#### Activity 1: Explain in Plain English

Learners explain LLMs, prompts, and agents without jargon.

#### Activity 2: Scenario Exit Ticket

Learners write one scenario for an LLM, one for RAG, and one for an agent.

### Checks for Understanding

1. How does an LLM generate a response?
2. Why do prompts matter?
3. What is grounding?
4. What makes an AI agent different from a basic chatbot?
