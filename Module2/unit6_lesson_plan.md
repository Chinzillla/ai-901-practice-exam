# Module 2: Introduction to Generative AI and Agents

## Unit 6: Module Assessment Prep

### Source

Microsoft Learn: [Module assessment](https://learn.microsoft.com/en-us/training/modules/fundamentals-generative-ai/8-knowledge-check)

### Unit Overview

This unit prepares learners for the Module 2 assessment without copying Microsoft Learn assessment items. It reviews LLMs, tokenization, embeddings, attention, prompts, grounding, and AI agents.

### Learning Objectives

By the end of this unit, learners should be able to:

- Review core LLM concepts.
- Distinguish prompts, completions, system prompts, and user prompts.
- Explain tokenization, embeddings, and attention at a high level.
- Identify agent components and tool types.
- Apply concepts to exam-style scenarios.

### Key Terms

| Term | Definition |
| --- | --- |
| Assessment prep | Original review practice before a module assessment. |
| Concept distinction | Separating related terms, such as token and embedding or system prompt and user prompt. |
| Scenario application | Choosing the correct concept based on a described requirement. |

### Lesson Flow

#### 1. Review LLM Concepts

Review how models use prompts, tokens, embeddings, transformers, attention, and next-token prediction.

#### 2. Review Prompting

Compare system prompts, user prompts, conversation history, and RAG.

#### 3. Review Agents

Review the agent model: LLM, instructions, and tools. Compare knowledge tools and action tools.

#### 4. Practice Scenarios

Use scenario questions that ask learners to identify the correct concept.

### Suggested Activities

#### Activity 1: Term Match

Learners match terms to definitions.

#### Activity 2: Scenario Review

Learners answer scenario questions and explain why each answer fits.

### Checks for Understanding

1. What is tokenization?
2. What are embeddings?
3. What is attention?
4. What is a system prompt?
5. What are the three components of an AI agent?
