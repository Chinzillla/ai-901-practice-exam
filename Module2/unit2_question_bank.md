### 1. A developer needs a model to draft an answer from a natural language prompt. Which model type is most relevant?

### 2. What is the main purpose of "large language model"?

A. a neural network design that uses attention to learn relationships in language

### 4. What distinguishes model training from inference?

A. Training learns patterns from data; inference uses the trained model to generate output.  
B. Training only formats UI controls, while inference scans paper forms.

### 7. Which two statements describe LLM behavior? Choose two.

A. They require a camera for every request
