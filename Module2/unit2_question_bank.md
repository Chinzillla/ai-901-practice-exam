## Unit 2 Question Bank: Large language models (LLMs)

These questions are authored from the unit's Microsoft Learn concepts and rewritten as exam-style practice items. Answer choices are intentionally paraphrased instead of copied from the lesson text.

### 1. A developer needs a model to draft an answer from a natural language prompt. Which model type is most relevant?

A. token  
B. transformer architecture  
C. completion  
D. A large language model  

**Answer:** D  
**Rationale:** The requirement matches the unit capability named in the correct option.

### 2. What is the main purpose of "large language model"?

A. a neural network design that uses attention to learn relationships in language  
B. the generated continuation or response the model produces from input  
C. a model trained on large text collections to predict and generate language  
D. a chunk of text the model processes, such as a word part or symbol  

**Answer:** C  
**Rationale:** "large language model" is a core idea in this unit, and the correct option paraphrases its role.

### 3. A learner asks why an LLM can generate a likely next phrase. What concept explains this behavior?

A. Object detection  
B. It predicts language patterns learned during training  
C. large language model  
D. Prompt engineering  

**Answer:** B  
**Rationale:** The correct option is the unit concept that best fits the stated requirement.

### 4. What distinguishes model training from inference?

A. Training learns patterns from data; inference uses the trained model to generate output.  
B. Training only formats UI controls, while inference scans paper forms.  
C. Inference changes all model weights for every user request.  
D. Training is only used for text-to-speech voices.  

**Answer:** A  
**Rationale:** The unit explains that trained models are later used to respond to prompts.

### 5. What should a learner remember about "token"?

A. a configuration task for the interface rather than the AI behavior  
B. a rule-based process that does not use model reasoning  
C. an access-control setting rather than the concept being defined  
D. a text unit the model processes, such as a word part or symbol  

**Answer:** D  
**Rationale:** "token" is tested by recognizing its purpose, not by memorizing wording from the page.

### 6. Why does model size matter when selecting an LLM?

A. Smaller models always outperform larger models on every task.  
B. Size is unrelated to cost or deployment choices.  
C. Larger models may handle broader tasks but can require more cost and resources.  
D. Model size determines only the color of the portal icon.  

**Answer:** C  
**Rationale:** AI-901 expects learners to reason about capability and deployment tradeoffs.

### 7. Which two statements describe LLM behavior? Choose two.

A. They require a camera for every request  
B. They only work with handwritten forms  
C. They guarantee factual answers without grounding  
D. They generate text based on learned language patterns  
E. They use prompts as input for responses  

**Answer:** D, E  
**Rationale:** LLMs are prompt-driven language generators, but they still need responsible use and validation.

### 9. Which three items are associated with LLM use? Choose three.

A. Bounding boxes  
B. Audio sampling rate

### 10. Which description best matches "transformer architecture"?
