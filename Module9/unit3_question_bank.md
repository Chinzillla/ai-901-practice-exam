# Module 9: Develop a Generative AI Chat App with Microsoft Foundry

## Unit 3 Question Bank: Choose an endpoint and SDK

These questions are authored from the unit's Microsoft Learn concepts and rewritten as exam-style practice items. Answer choices are intentionally paraphrased instead of copied from the lesson text.

---

### 1. A JavaScript app needs to call a deployed model in Foundry. What two things must the developer choose?

A. client  
B. Prompt engineering  
C. Object detection  
D. An endpoint and a compatible SDK or API  

**Answer:** D  
**Rationale:** The requirement matches the unit capability named in the correct option.

---

### 2. What is the main purpose of "endpoint"?

A. a programming interface for sending requests and receiving responses  
B. application code configured to communicate with an endpoint  
C. the target address for calling a deployed model  
D. a library that simplifies calls to AI services from code  

**Answer:** C  
**Rationale:** "endpoint" is a core idea in this unit, and the correct option paraphrases its role.

### 3. A Python developer wants helper classes for requests instead of raw HTTP calls. What should they use?

A. Speech synthesis  
B. An SDK  
C. endpoint  
D. API  

**Answer:** B  
**Rationale:** The correct option is the unit concept that best fits the stated requirement.

### 4. Why choose an SDK instead of hand-coding every HTTP request?

A. An SDK simplifies authentication, request construction, and response handling.  
B. SDKs remove all need for endpoints.  
C. SDKs only work for image cropping.  
D. SDKs make model evaluation unnecessary.  

**Answer:** A  
**Rationale:** SDKs are developer tools for integrating services into apps.

### 5. What should a learner remember about "SDK"?

A. a configuration task for the interface rather than the AI behavior  
B. a rule-based process that does not use model reasoning  
C. an access-control setting rather than the concept being defined  
D. library that simplifies calls to AI services from code  

**Answer:** D  
**Rationale:** "SDK" is tested by recognizing its purpose, not by memorizing wording from the page.

### 6. What should a client app be configured with before sending requests?

A. A handwritten transcript.  
B. No authentication details at all.  
C. Endpoint information and credentials for the deployed model or service.  
D. Only a screenshot of the model card.  

**Answer:** C  
**Rationale:** Client applications need service access configuration.

### 7. Which two items are needed for most SDK-based model calls? Choose two.

A. Image filter kernel  
B. Speech phoneme table  
C. Portal navigation setting  
D. Callable service address  
E. Credential  

**Answer:** D, E  
**Rationale:** The app needs a target and authorized access.

### 8. An app sends prompts to a specific deployed model URL. What is the URL called?

### 9. Which three steps belong in a lightweight client app workflow? Choose three.

A. Ignore errors forever  
B. Publish secrets in source control  
C. Use only manual copy and paste  
D. Create a client  
E. Send a prompt or request  
F. Read the model response  

**Answer:** D, E, F  
**Rationale:** A client app configures access, sends requests, and handles responses.
