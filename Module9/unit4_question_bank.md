# Module 9: Develop a Generative AI Chat App with Microsoft Foundry

## Unit 4 Question Bank: Generate responses with the Responses API

These questions are authored from the unit's Microsoft Learn concepts and rewritten as exam-style practice items. Answer choices are intentionally paraphrased instead of copied from the lesson text.

---

### 1. A developer sends a prompt and receives generated text through a modern model API. Which API concept is central?

A. input  
B. tool support  
C. Prompt engineering  
D. Responses API request and output  

**Answer:** D  
**Rationale:** The requirement matches the unit capability named in the correct option.

---

### 2. What is the main purpose of "Responses API"?

A. the generated result returned by the model  
B. the ability for a model request to use configured tools when available  
C. an API for sending model input and receiving generated responses  
D. the prompt or content provided to the model request  

**Answer:** C  
**Rationale:** "Responses API" is a core idea in this unit, and the correct option paraphrases its role.

---

### 3. The app passes the user's question to the model. What part of the request is this?

A. Object detection  
B. Input capability  
C. Responses API  
D. output  

**Answer:** B  
**Rationale:** The correct option is the unit concept that best fits the stated requirement.

---

### 4. What is the practical role of the Responses API in a chat app?

A. It provides a structured way for code to send prompts and handle generated results.  
B. It only stores image files.  
C. It replaces every need for model deployment.  
D. It is used only for manual portal testing.  

**Answer:** A  
**Rationale:** The unit focuses on calling a model from application code.

---

### 5. What should a learner remember about "input"?

A. a configuration task for the interface rather than the AI behavior  
B. a rule-based process that does not use model reasoning  
C. an access-control setting rather than the concept being defined  
D. prompt or content provided to the model request  

**Answer:** D  
**Rationale:** "input" is tested by recognizing its purpose, not by memorizing wording from the page.

### 6. Why should application code inspect the model response object?

A. Inspecting responses disables authentication.  
B. The model answer is always printed by the portal only.  
C. The generated answer and related details must be extracted from the returned response.  
D. The response object never contains output.  

**Answer:** C  
**Rationale:** Client code must parse and use the response.

---

### 7. Which two items are part of a basic Responses API interaction? Choose two.

A. Scan a paper form by default  
B. Record microphone audio only  
C. Change monitor resolution  
D. Send model input  
E. Read generated output  

**Answer:** D, E  
**Rationale:** The core request-response pattern is input to output.

### 8. The app reads the generated answer returned by the model. What is this?

**Answer:** A  
**Rationale:** The scenario points to the unit capability described by the correct option.

### 9. Which three concerns matter when building a lightweight Responses API client? Choose three.

A. Ignore credentials  
B. Assume no errors can occur  
C. Use unrelated vision labels  
D. Configure the client

**Answer:** D, E, F  
**Rationale:** A working client needs configuration, request input, and response handling.
