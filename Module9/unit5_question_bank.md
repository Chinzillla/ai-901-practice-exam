# Module 9: Develop a Generative AI Chat App with Microsoft Foundry

## Unit 5 Question Bank: Generate responses with the ChatCompletions API

These questions are authored from the unit's Microsoft Learn concepts and rewritten as exam-style practice items. Answer choices are intentionally paraphrased instead of copied from the lesson text.

---

### 1. A chat app sends a list of system, user, and assistant messages to generate the next reply. Which API style is this?

A. system message  
B. user message  
C. assistant message  
D. ChatCompletions API  

**Answer:** D  
**Rationale:** The requirement matches the unit capability named in the correct option.

### 2. What is the main purpose of "ChatCompletions API"?

A. a message containing the user's request  
B. a generated response or prior assistant turn in the conversation  
C. an API pattern that uses conversation messages to generate chat responses  
D. a message that sets assistant behavior or role  

**Answer:** C  
**Rationale:** "ChatCompletions API" is a core idea in this unit, and the correct option paraphrases its role.

### 3. A developer wants the assistant to behave as a helpful math tutor throughout a conversation. Which message type should set that role?

### 4. What makes the ChatCompletions pattern conversation-oriented?

A. It sends role-based messages that represent the conversation context.  
B. It never includes user input.

**Answer:** A  
**Rationale:** The unit presents chat completions as role-message based.

### 5. What should a learner remember about "system message"?

### 6. Why include conversation history in a chat completion request?

B. History changes the endpoint URL automatically.

**Answer:** C  
**Rationale:** Chat models use prior messages as conversational context.
