These questions are authored from the unit's Microsoft Learn concepts and rewritten as exam-style practice items. Answer choices are intentionally paraphrased instead of copied from the lesson text.

### 1. A developer wants to test model responses before writing code. Which Foundry feature should they use?

A. system message  
B. prompt iteration  
C. parameters  
D. The model playground  

**Answer:** D  
**Rationale:** The requirement matches the unit capability named in the correct option.

### 2. What is the main purpose of "model playground"?

A. revising prompts and settings based on observed responses  
B. settings that influence model response style or behavior  
C. an interactive portal experience for testing prompts and model responses  
D. instructions that shape assistant behavior across the conversation  

**Answer:** C  
**Rationale:** "model playground" is a core idea in this unit, and the correct option paraphrases its role.

### 3. The assistant should always answer as a concise travel advisor. Where should that behavior be set?

### 4. How does playground testing help before SDK development?
