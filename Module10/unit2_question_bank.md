# Module 10: Develop Generative AI Apps that Use Tools

## Unit 2 Question Bank: What are tools?

These questions are authored from the unit's Microsoft Learn concepts and rewritten as exam-style practice items. Answer choices are intentionally paraphrased instead of copied from the lesson text.

---

### 1. A model needs current information that was not in its training data. What should the app provide?

A. tool call  
B. action tool  
C. Prompt engineering  
D. A knowledge tool  

**Answer:** D  
**Rationale:** This scenario matches What are tools? as described in the unit.

---

### 2. What is the main purpose of "tool"?

A. a tool that retrieves information from sources such as search or files  
B. a tool that performs an operation such as calling a function or updating a system  
C. an external capability a model can use to retrieve information or perform a task  
D. the model's request to use a configured tool during a response  

**Answer:** C  
**Rationale:** "tool" is a core idea in this unit, and the correct option paraphrases its role.

---

### 3. A model decides to call a configured function to get order status. What is this interaction?

A. Speech synthesis  
B. A tool call  
C. knowledge tool  
D. Object detection  

**Answer:** B  
**Rationale:** The correct option is the unit concept that best fits the stated requirement.

---

### 4. Why are tools useful with generative AI models?

A. They let models use external information or actions instead of relying only on generated text.  
B. Tools stop models from using prompts.  
C. Tools are only visual icons in the portal.  
D. Tools remove all need for validation.  

**Answer:** A  
**Rationale:** The unit introduces tools as model extensions.

---

### 5. What should a learner remember about "tool call"?

A. a process for changing the visual theme of the development portal  
B. a manual-only activity that does not use an AI model  
C. a security shortcut that removes the need for access controls  
D. the model's request to use a configured tool during a response as described in this unit  

**Answer:** D  
**Rationale:** "tool call" is tested by recognizing its purpose, not by memorizing wording from the page.

---

### 6. What should a developer define when exposing a function as a tool?

A. A random endpoint with no schema.  
B. A ban on all model instructions.  
C. Clear tool purpose, input parameters, and expected output.  
D. Only the button color.  

**Answer:** C  
**Rationale:** Tools need clear definitions so the model can use them correctly.

---

### 7. Which two are examples of tool use? Choose two.

A. Ignoring user intent  
B. Guessing data without sources  
C. Changing screen brightness only  
D. Searching external information  
E. Calling a function to complete a task  

**Answer:** D, E  
**Rationale:** Tools support information retrieval and actions.

---

### 8. An assistant creates a ticket in another system after confirming details. What kind of tool is being used?

A. An action tool  
B. Information extraction  
C. Model evaluation  
D. Choose a model only because it has the largest name recognition.  

**Answer:** A  
**Rationale:** The scenario points to the unit capability described by the correct option.

---

### 9. Which three tool-related concepts should learners recognize? Choose three.

A. Image resolution only  
B. Phoneme cadence  
C. Keyboard shortcut  
D. Tool definition  
E. Tool call for this scenario  
F. Tool result  

**Answer:** D, E, F  
**Rationale:** Tool workflows involve defining tools, calling them, and using returned results.

---

### 10. Which description best matches "knowledge tool"?

A. Use image analysis even when the input is only plain text.  
B. Store sensitive data without reviewing privacy requirements.  
C. a tool that retrieves information from sources such as search or files as described in this unit  
D. Skip testing because AI outputs are always deterministic.  

**Answer:** C  
**Rationale:** The correct answer paraphrases how the unit explains "knowledge tool".
