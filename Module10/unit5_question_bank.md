# Module 10: Develop Generative AI Apps that Use Tools

## Unit 5 Question Bank: Use the file_search tool

These questions are authored from the unit's Microsoft Learn concepts and rewritten as exam-style practice items. Answer choices are intentionally paraphrased instead of copied from the lesson text.

### 1. An assistant must answer questions from a company's uploaded policy PDFs. Which tool should it use?

A. vector store  
B. retrieval  
C. grounded answer  
D. file_search  

**Answer:** D  
**Rationale:** The requirement matches the unit capability named in the correct option.

### 2. What is the main purpose of "file_search"?

A. finding relevant passages or documents for a model request  
B. a response based on retrieved file content  
C. a tool for retrieving relevant content from uploaded or indexed files  
D. an index that supports semantic retrieval over file content  

**Answer:** C  
**Rationale:** "file_search" is a core idea in this unit, and the correct option paraphrases its role.

### 3. A developer indexes documents so the model can retrieve relevant passages later. What is being prepared?

**Answer:** B  
**Rationale:** The correct option is the unit concept that best fits the stated requirement.

### 4. How is file_search different from web_search?

A. file_search retrieves from provided files; web_search retrieves from the public web.  
B. file_search only runs Python.  
C. web_search only searches uploaded PDFs.

**Rationale:** The unit distinguishes retrieval sources.

### 5. What should a learner remember about "vector store"?

A. a configuration task for the interface rather than the AI behavior

### 6. Why are files indexed before retrieval?
