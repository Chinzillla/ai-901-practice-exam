# Module 10: Develop Generative AI Apps that Use Tools

## Unit 6 Question Bank: Use the function tool

These questions are authored from the unit's Microsoft Learn concepts and rewritten as exam-style practice items. Answer choices are intentionally paraphrased instead of copied from the lesson text.

---

### 1. A model needs to check order status by calling application code. Which tool type fits?

A. function schema  
B. function arguments  
C. function result  
D. Function tool  

**Answer:** D  
**Rationale:** The requirement matches the unit capability named in the correct option.

---

### 2. What is the main purpose of "function tool"?

A. structured values the model provides when requesting a function call  
B. the data returned by application code after the function runs  
C. a tool definition that lets a model request application-defined code or operations  
D. the name, description, and parameters that tell the model how to call a function  

**Answer:** C  
**Rationale:** "function tool" is a core idea in this unit, and the correct option paraphrases its role.

---

### 3. A developer defines required parameters such as orderId and customerEmail. What is being defined?

A. Speech synthesis  
B. Function schema capability  
C. Prompt engineering  
D. Object detection  

**Answer:** B  
**Rationale:** The correct option is the unit concept that best fits the stated requirement.

---

### 4. How is a function tool different from code_interpreter?

A. A function tool calls developer-defined app logic; code_interpreter runs model-generated Python in a sandbox.  
B. Function tools only retrieve web pages.  
C. code_interpreter requires developer-defined business functions.  
D. Both tools are only for OCR.  

**Answer:** A  
**Rationale:** The unit separates app-defined function calls from sandboxed Python execution.

---

### 5. What should a learner remember about "function schema"?

A. a configuration task for the interface rather than the AI behavior  
B. a rule-based process that does not use model reasoning  
C. an access-control setting rather than the concept being defined  
D. name, description, and parameters that tell the model how to call a function  

**Answer:** D  
**Rationale:** "function schema" is tested by recognizing its purpose, not by memorizing wording from the page.

---

### 6. Why should function schemas be clear?

A. Unclear schemas improve tool selection.  
B. Schemas only change response color.  
C. The model needs accurate names, descriptions, and parameters to call the right function.  
D. Schemas should hide all parameters from the model.  

**Answer:** C  
**Rationale:** Clear schemas help the model choose and call functions correctly.

---

### 7. Which two items belong in a function tool definition? Choose two.

A. Image brightness  
B. Voice pitch  
C. Unrelated web headline  
D. Function description  
E. Parameter schema  

**Answer:** D, E  
**Rationale:** The model needs to know what the function does and what arguments it requires.

---

### 8. The model supplies orderId: 12345 when requesting a tool call. What are these values?

A. Function arguments capability  
B. Information extraction  
C. Model evaluation  
D. Choose a model based on popularity instead of task fit.  

**Answer:** A  
**Rationale:** The scenario points to the unit capability described by the correct option.

---

### 9. Which three steps are part of a function tool workflow? Choose three.

A. Let the model access secrets directly  
B. Ignore argument validation  
C. Assume no application code is needed  
D. Define the function  
E. Receive model-proposed arguments  
F. Return the function result to the model  

**Answer:** D, E, F  
**Rationale:** Function tools connect model reasoning with application-controlled execution.

### 10. Which description best matches "function arguments"?

A. Use a visual analysis capability when the input is only text.  
B. Handle sensitive data without the required access controls.  
C. arguments supplied by the model when requesting a function call  
D. Rely on one successful trial instead of broader evaluation.  

**Answer:** C  
**Rationale:** The correct answer paraphrases how the unit explains "function arguments".
