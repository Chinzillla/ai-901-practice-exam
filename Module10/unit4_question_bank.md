# Module 10: Develop Generative AI Apps that Use Tools

## Unit 4 Question Bank: Use the web_search tool

These questions are authored from the unit's Microsoft Learn concepts and rewritten as exam-style practice items. Answer choices are intentionally paraphrased instead of copied from the lesson text.

---

### 1. A user asks for the latest product release date. Which tool should the model use?

A. fresh information  
B. source grounding  
C. search query  
D. web_search  

**Answer:** D  
**Rationale:** The requirement matches the unit capability named in the correct option.

---

### 2. What is the main purpose of "web_search"?

A. using retrieved sources to support generated answers  
B. the question or keywords sent to the search tool  
C. a tool for retrieving current information from the web  
D. information that may have changed after a model was trained  

**Answer:** C  
**Rationale:** "web_search" is a core idea in this unit, and the correct option paraphrases its role.

---

### 3. A model needs information that may have changed this week. What is the main reason to use web search?

A. Speech synthesis  
B. To retrieve fresh information  
C. Prompt engineering  
D. Object detection  

**Answer:** B  
**Rationale:** The correct option is the unit concept that best fits the stated requirement.

---

### 4. When is web_search a better fit than file_search?

A. Use web_search for public current web information; use file_search for indexed private files.  
B. web_search is only for Python code.  
C. file_search is used for live news.  
D. Both tools ignore retrieved content.  

**Answer:** A  
**Rationale:** The unit distinguishes web retrieval from file-based retrieval.

---

### 5. What should a learner remember about "fresh information"?

A. a configuration task for the interface rather than the AI behavior  
B. a rule-based process that does not use model reasoning  
C. an access-control setting rather than the concept being defined  
D. current information that may not be in after a model was trained  

**Answer:** D  
**Rationale:** "fresh information" is tested by recognizing its purpose, not by memorizing wording from the page.

---

### 6. Why should a model form a clear search query?

A. Queries only affect image generation.  
B. Clear queries prevent source use.  
C. A focused query helps retrieve relevant information for the user's request.  
D. Search tools work best with empty queries.  

**Answer:** C  
**Rationale:** Search quality depends on the query and retrieved sources.

---

### 7. Which two requests are good candidates for web_search? Choose two.

A. Analyze an uploaded CSV with Python  
B. Search a private file index  
C. Convert text to speech  
D. Find current documentation details  
E. Check recent public information  

**Answer:** D, E  
**Rationale:** web_search is for current public web retrieval.

---

### 8. An answer should cite or reflect retrieved web pages instead of relying only on memory. What practice is this?

A. Grounding with web sources  
B. Information extraction  
C. Model evaluation  
D. Choose a model based on popularity instead of task fit.  

**Answer:** A  
**Rationale:** The scenario points to the unit capability described by the correct option.

---

### 9. Which three practices support useful web_search answers? Choose three.

A. Invent unsupported facts  
B. Ignore source freshness  
C. Use unrelated private files  
D. Search for current information  
E. Use relevant retrieved sources  
F. Answer the user's specific question  

**Answer:** D, E, F  
**Rationale:** Good web-search responses are focused, current, and grounded.

### 10. Which description best matches "source grounding"?

A. Use a visual analysis capability when the input is only text.  
B. Handle sensitive data without the required access controls.  
C. use of retrieved sources to support generated answers  
D. Rely on one successful trial instead of broader evaluation.  

**Answer:** C  
**Rationale:** The correct answer paraphrases how the unit explains "source grounding".
