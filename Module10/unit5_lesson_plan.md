# Module 10: Develop Generative AI Apps that Use Tools

## Unit 5: Use the file_search tool

### Source

[Microsoft Learn unit](https://learn.microsoft.com/en-us/training/modules/use-generative-ai-tools/05-file-search)

### Unit Overview

Learners explore file_search for grounding responses in uploaded, indexed documents such as policies, manuals, contracts, and knowledge bases.

### Learning Objectives

- describe file_search capabilities
- explain vector store usage
- identify document-grounded scenarios
- recognize limitations of retrieval quality

### Key Terms

| Term | Beginner-Friendly Definition |
| --- | --- |
| file_search | A tool that retrieves relevant passages from uploaded indexed files. |
| Vector store | A searchable collection of embedded document chunks. |
| Semantic retrieval | Finding relevant content by meaning rather than exact words only. |
| Citations | References or matched results that help trace where an answer came from. |
| Re-indexing | Updating a search index so changed documents become searchable. |

### Lesson Flow

#### 1. Use cases

- file_search fits internal policies, manuals, contracts, and domain-specific knowledge.
- It is strongest when trusted documents contain the needed answer.

#### 2. Process

- Upload files, index them into a vector store, include file_search with vector store IDs, retrieve passages, and generate a grounded answer.

#### 3. Quality controls

- Clean source files, focused prompts, scoped vector stores, and human review improve reliability.

### Suggested Activities

- Ask learners to summarize the unit in one sentence using beginner-friendly language.
- Give learners a short workplace scenario and have them choose the correct concept, tool, strategy, or responsibility.
- Have learners identify one exam clue word from the unit, such as current, uploaded, benchmark, grounded, fine-tuned, guardrail, or endpoint.
- Pair learners to explain why one tempting alternative is not the best answer.

### Checks for Understanding

- What problem does this unit's main concept solve?
- Which keyword in a scenario would point you toward this concept?
- What is one limitation or caution learners should remember?
- How would this concept appear in a simple AI-901-style business scenario?

### Assessment

Use the accompanying question bank to check recognition, scenario selection, vocabulary, and common misconceptions. Learners should be able to explain not only the correct answer, but why the distractors are less appropriate.

### Teaching Notes

- Keep the explanation practical and scenario-based.
- Emphasize the difference between similar concepts instead of asking learners to memorize product names only.
- Connect the unit to AI-901 readiness by asking, "What would the exam scenario be asking me to choose?"
- Reinforce that secure, responsible, and measured development choices matter in generative AI apps.

### Estimated Duration

45-60 minutes
